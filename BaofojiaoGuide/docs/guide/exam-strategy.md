# 通用考试策略

以下策略适用于 CIE 所有科目与试卷，考前务必牢记。

---

## 答题前（前 2-3 分钟）

1. **通览全卷** — 快速翻看所有题目，了解题型分布与难度
2. **标记优先级** — 圈出你最有把握的题目，优先作答
3. **分配时间** — 根据每题分值估算作答时间，写在试卷上

---

## 答题中

### 一般原则

- **先易后难** — 先做有把握的题，建立信心，确保基础分到手
- **展示过程** — CIE 评分看重 method marks，即使最后答案错了，过程正确仍有分
- **写相关公式** — 如果卡住了，把相关的公式写下来，可能触发思路或拿到方法分
- **关键词答题** — 使用 Mark Scheme 中的高频关键词，确保踩中得分点
- **不留空白** — 实在不会的题也要写点相关的内容，零分和部分分差别很大

### 时间管理

- **不要在一道题上耗尽时间** — 超过预算时间就跳到下一题
- **留 5 分钟检查** — 至少留出时间复查计算和单位
- **难题先跳过** — 标记后继续做后面的题，回头有时间再补

### 不同题型策略

| 题型 | 策略 |
|------|------|
| **选择题** | 排除法优先，不确定的标记后跳过，留到最后蒙 |
| **结构化问答** | 先读问题再读题干，带着问题找答案；分点作答，每点一行 |
| **计算题** | 写出公式 → 代入数值 → 写出中间结果 → 最终答案（带单位） |
| **实验题** | 明确自变量、因变量、控制变量；画图注意标签和单位 |
| **编程题** | 先写伪代码理清逻辑，再转换为目标语言 |

---

## 答题后（最后 5 分钟）

1. **检查答案格式** — 单位、有效数字、小数点是否规范
2. **补全遗漏** — 是否漏答了小题或 sub-part
3. **检查计算** — 快速重算关键步骤，防止粗心错误
4. **不要轻易改答案** — 除非确认原答案是错的，第一直觉通常更可靠

---

## 考前提醒

- **带齐工具** — 计算器（备用电池）、直尺、铅笔、橡皮
- **公式表** — 确认哪些公式会提供，哪些需要自己记
- **睡好** — 前一晚保证睡眠，比多刷一套题更有用
