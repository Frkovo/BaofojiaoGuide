# 抱佛脚 Guide

> **CIE IGCSE / AS / A Level 考前突击复习指南**

本站专为 CIE 考生打造，提供**科目概览、试卷策略、逐主题精讲、评分关键词分析、常见错误避坑、考前速查清单**等全方位备考资源，助你临时抱佛脚也能抱得稳、抱得准。

## 涵盖科目

| 科目 | 代码 | 覆盖试卷 |
|------|------|---------|
| **物理学** | 9702 | Paper 4（结构化问答）、Paper 5（实验设计与分析） |
| **进阶数学** | 9231 | Paper 2（纯数）、Paper 3（力学） |

每个试卷均包含：考纲解析 → 题型拆解 → 时间管理 → 常见陷阱 → 评分关键词 → 考前速查

---

## 快速开始

1. 从左侧**侧边栏**选择科目与试卷
2. 先阅读该试卷的 **Strategy（策略）** 页面，了解整体考试思路
3. 逐一攻克 **Topics（主题）**，每个主题都有方法总结与评分模式分析
4. 考前翻阅 **Last-Minute Guide（考前速查）**，快速回顾要点
