# 考纲要求

## 15.2 Boolean Algebra and Logic Circuits
| 编号 | 要求 |
|------|------|
| 15.2.1 | Construct and complete **truth tables** for given logic circuits |
| 15.2.2 | Derive and simplify **Boolean expressions** from logic circuits |
| 15.2.3 | Apply **De Morgan's laws**：$$\overline{A \cdot B} = \overline{A} + \overline{B}$$, $$\overline{A + B} = \overline{A} \cdot \overline{B}$$ |
| 15.2.4 | Simplify Boolean expressions using **commutative, associative, distributive** laws |
| 15.2.5 | Use **Karnaugh maps** to simplify Boolean expressions (2, 3, 4 variables) |
| 15.2.6 | Derive **sum-of-products (SOP)** form from truth tables |
| 15.2.7 | Describe the function of **SR flip-flop** and **JK flip-flop** |
| 15.2.8 | Complete truth tables for **flip-flops** (SR, JK) |
| 15.2.9 | Design and analyze **half adder** and **full adder** circuits |
