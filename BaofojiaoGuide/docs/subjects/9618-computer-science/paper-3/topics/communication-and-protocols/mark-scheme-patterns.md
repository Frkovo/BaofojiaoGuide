# MS 模式总结

## 标记说明
| 标记 | 含义 | 要求 |
|------|------|------|
| **B1** | Behaviour mark | 正确的独立陈述（Describe/State 题） |
| **M1** | Method mark | Labelled diagram 或计算步骤 |

## 常见 MS 短语

### TCP/IP Layers
- "**B1**：Application layer — HTTP, FTP, SMTP etc."
- "**B1**：Transport layer — TCP/UDP, end-to-end"
- "**B1**：Internet layer — IP addressing, routing"
- "**B1**：Link layer — physical transmission"

### Protocols
- "**B1**：HTTP — transfer of web pages"
- "**B1**：FTP — file transfer"
- "**B1**：SMTP — sending email"
- "**B1**：POP3 — downloading email to local client"
- "**B1**：BitTorrent — peer-to-peer file sharing"

### Packet vs Circuit Switching
- "**B1**：circuit switching — dedicated path established"
- "**B1**：packet switching — data split into packets"
- "**B1**：advantage of packet switching — efficient bandwidth use"
- "**B1**：disadvantage of circuit switching — inefficient for burst data"

## Examiner Tips
- **Describe** 题：需要有完整的句子，不能只是关键词
- **Compare** 题：需要 **both** sides（A 和 B 都要提到）
- **Advantage** 需要带理由（Why is it better?），不能只说 "fast"
