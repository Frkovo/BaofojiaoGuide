# 考纲要求

## 14.1 Protocols
| 编号 | 要求 |
|------|------|
| 14.1.1 | Describe the purpose of **TCP/IP** (Transmission Control Protocol / Internet Protocol) |
| 14.1.2 | Describe the **four layers** of TCP/IP: Application, Transport, Internet, Link |
| 14.1.3 | Describe the purpose of **HTTP / HTTPS** (web page transfer) |
| 14.1.4 | Describe the purpose of **FTP** (file transfer) |
| 14.1.5 | Describe the purpose of **SMTP** (send email) |
| 14.1.6 | Describe the purpose of **POP3 / IMAP** (retrieve email) |
| 14.1.7 | Describe the purpose of **BitTorrent** (peer-to-peer file sharing) |

## 14.2 Circuit Switching, Packet Switching, Routers
| 编号 | 要求 |
|------|------|
| 14.2.1 | Describe **circuit switching** (dedicated path, telephone network) |
| 14.2.2 | Describe **packet switching** (data split into packets, independent routing) |
| 14.2.3 | Compare advantages and disadvantages of circuit and packet switching |
| 14.2.4 | Describe the function of a **router** (reads IP address, routing table, forwards) |
