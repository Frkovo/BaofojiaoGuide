## 高频记忆点
- TCP/IP 四层（上到下）：Application → Transport → Internet → Link
- Packet switching：高效利用带宽、容错；Circuit switching：低延迟、保证带宽
- SMTP = 发邮件，POP3/IMAP = 收邮件
