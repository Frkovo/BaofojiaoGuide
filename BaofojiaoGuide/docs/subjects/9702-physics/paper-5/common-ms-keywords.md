# MS 关键词速查

## Q1 实验设计

| MS 原句 | 中文含义 | 分值 |
|---------|---------|------|
| `independent variable is X` | 自变量是 X | **1** |
| `dependent variable is Y` | 因变量是 Y | **1** |
| `vary X and measure Y` | 改变 X，测量 Y | **1** |
| `keep Z constant` | 保持 Z 不变 | **1** |
| `labelled diagram of workable experiment` | 画出可操作的实验装置图并标注 | **1** |
| `repeat measurements and average` | 重复测量取平均值 | **1** |
| `plot a graph of Y against X` | 画 Y 对 X 的图 | **1** |
| `straight line passing through the origin` | 直线过原点 | **1** |
| `straight line (not passing through the origin)` | 直线不过原点 | **1** |
| `relationship valid if a straight line is produced` | 如果得到直线则关系成立 | **1** |
| `safety goggles` / `cushion/sand box` | 护目镜/缓冲装置 | **1** |
| `use micrometer/calipers` | 使用千分尺/卡尺 | **1** |
