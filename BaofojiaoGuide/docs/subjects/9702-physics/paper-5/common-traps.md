# 常见陷阱

### Trap 1: 对数坐标的单位处理

- **问题**: 忘记在对数前将量除以单位
- **后果**: 单位错误，$\ln (V/\text{V})$ 中的 V 不能省略
- **对策**: 始终写 `$\ln (X/\text{unit})$`

### Trap 2: 线性化方向错误

- **问题**: $y = ax^n$ 想要线性化时，选了 $\ln x$ 对 $\ln y$ 还是 $\lg x$ 对 $\lg y$ 都可以，但必须保持一致
- **对策**: 两种都可以，但 ln 和 lg 不能混用

### Trap 3: Worst Acceptable Line 的误解

- **问题**: 以为 WAL 一定要穿过所有点的中心
- **正确**: WAL 是最陡或最缓的直线，但要通过所有 error bars
- **技巧**: 先把 best fit 画好，然后试最陡和最缓两条线

### Trap 4: 梯度符号

- **问题**: 计算 $C$ 时忘了负号，因为 $V = V_0 e^{-t/RC}$ 中梯度 $= -1/RC$
- **对策**: 写下表达式后检查正负号

### Trap 5: 控制变量的混淆

- **问题**: 把常数 (如 $\rho, g$) 和控制变量混淆
- **正确**: 常数是方程中给定的固定值，控制变量是实验中可以变化的量

### Trap 6: 省略 diagram

- **问题**: 以为画图不重要
- **后果**: 漏掉 1 分
- **对策**: 即使画得不好，也要画并标注 3-4 个关键部件

### Trap 7: 最大最小法求不确定度

- **问题**: 不知道最大最小法也接受
- **正确**: 不确定度 = max - value 或 value - min

### Trap 8: 公式表中的数据

- **问题**: 没有使用题目给定的 $R$ 值
- **对策**: 仔细读题，所有给定常数必须使用

### Trap 9: 忽略 error bars 就画 WAL

- **问题**: WAL 没有通过所有 error bars
- **对策**: 先画 error bars，再画 WAL

### Trap 10: 实验中的单位换算

- **问题**: 单位的 10 的幂次错误，如 $R = 39\ \text{k}\Omega$ 写成 $39\ \Omega$
- **对策**: 代入前检查单位换算
