## 考纲要求

- Use gradient and y-intercept expressions to calculate constants
- Include correct units
- Propagate uncertainties to final answers
- Draw conclusions from experimental data
- Make predictions and determine values

## 常见题型

- [One Constant from Gradient](./question-types.md#question-type-1-calculate-one-constant-from-gradient) — 从 gradient 求常数
- [Two Constants from Gradient and Intercept](./question-types.md#question-type-2-calculate-two-constants-from-gradient-and-intercept) — 同时从 gradient 和 intercept 求两个常数
- [Extension Calculations](./question-types.md#question-type-3-extension-calculations) — 用已求常数进一步计算
- [Drawing Conclusions](./question-types.md#question-type-4-drawing-conclusions) — 实验结果分析与结论

## 核心流程

- Gradient expression $\rightarrow$ calculate constant 1
- y-intercept expression $\rightarrow$ calculate constant 2
- Check units are dimensionally correct
- Find uncertainty using error propagation

## 常见错误

- 忘记在常数后加单位
- 公式代入时符号错误（特别是负号）
- 未使用题目给的已知值（如 $k = 1.38 \times 10^{-23}$）
