# 考纲要点

## 实验设计四个部分

### Part 1: Defining the Problem（2 分）

- **B1**: 正确识别 independent variable 和 dependent variable
- **B1**: 列出至少 2 个 control variables

### Part 2: Methods of Data Collection（5 分）

- **B1**: 画出有标注的实验装置图
- **B1**: 描述如何改变 independent variable
- **B1**: 描述如何测量 dependent variable
- **B1**: 描述如何保持控制变量恒定
- **B1**: 描述测量范围和测量次数

### Part 3: Method of Analysis（3 分）

- **B1**: 说明要画什么类型的图（linearized form）
- **B1**: 说明如果关系成立，图应当呈现什么结果（straight line）
- **B1**: 说明如何从图中求出常数（gradient / intercept 表达式）

### Part 4: Additional Detail and Safety（5 分）

- **D1 - D5**: 每个额外的细节得 1 分
  - 重复测量 and 取平均值
  - 精确测量某个小量
  - 额外的仪器使用
  - 安全措施
  - 其他合理的细节

## 评分总览

| 部分 | 内容 | 分值 |
|------|------|------|
| Defining the problem | IV, DV, CVs | 2 |
| Methods of data collection | 装置图 + 步骤 | 5 |
| Method of analysis | 图像 + 常数 | 3 |
| Additional detail / safety | 5 个细节 | 5 |
| **Total** | | **15** |
