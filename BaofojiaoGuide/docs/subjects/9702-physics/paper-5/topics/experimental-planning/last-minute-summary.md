# 考前速通

## Must-Know Linearization Formulas

| 原始关系 | 线性化形式 | gradient | y-intercept |
|---------|-----------|----------|-------------|
| $y = ax^n$ | $\lg y = \lg a + n \lg x$ | $n$ | $\lg a$ |
| $y = ae^{kx}$ | $\ln y = \ln a + kx$ | $k$ | $\ln a$ |
| $y = a + \frac{b}{x}$ | $y$ against $\frac{1}{x}$ | $b$ | $a$ |
| $y = \frac{a}{x}$ | $y$ against $\frac{1}{x}$ | $a$ | $0$ |
| $y = a\sqrt{x} + b$ | $y$ against $\sqrt{x}$ | $a$ | $b$ |
| $y^2 = ax$ | $y^2$ against $x$ | $a$ | $0$ |

## Red Flags

| 信号 | 可能的问题 |
|------|-----------|
| 看到 $\lg$ 或 $\ln$ | 必须线性化为对数形式 |
| 公式中有 $\pi$ | 需要测量直径 $d$ |
| 提到 spring | 小心 safety — snapping / falling masses |
| 提到 electrical | 小心 overheating — 断开电源 |
| 看到 $^{\frac{3}{2}}$ 等指数 | 可能需要对 $\lg$ 作图 |
| 多个控制变量 | 至少列 2 个 |
| 题目给了 $n$ 或 $k$ | 通常需要求这些常数 |

## Quick Checklist

### Defining the Problem（2 分）
- [ ] Independent variable 明确写出
- [ ] Dependent variable 明确写出
- [ ] At least 2 control variables named

### Methods of Data Collection（5 分）
- [ ] Labelled diagram drawn
- [ ] How to vary IV described
- [ ] How to measure DV described
- [ ] How to keep CVs constant
- [ ] Range and number of readings stated

### Method of Analysis（3 分）
- [ ] Graph type specified（x and y axis）
- [ ] Expected result stated（straight line）
- [ ] Constant expressed in terms of gradient / intercept

### Additional Detail（5 分）
- [ ] Repeat measurements
- [ ] Safety precaution
- [ ] Precise measurement of small quantity
- [ ] Extra parameter measurement
- [ ] Other relevant detail

:::tip

先拿基础分（**B1** 共 10 分），再追细节分（**D1** 共 5 分）。确保每个 **B1** 对应点都覆盖。

:::
