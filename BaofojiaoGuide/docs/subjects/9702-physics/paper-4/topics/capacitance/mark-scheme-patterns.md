# Capacitance — MS 模式

## 定义题模式

| 术语 | 分值 | 标准答案 |
|------|------|---------|
| capacitance | M1+A1 | charge per unit potential difference + charge on one plate + p.d. across plates |
| time constant | B1 | $RC$ / time for $V$ to fall to $1/e$ of initial value |

## 计算题模式

| 计算类型 | 标记 | 公式 |
|---------|------|------|
| 并联电容 | C1+A1 | $C = C_1 + C_2$ |
| 串联电容 | C1+A1 | $1/C = 1/C_1 + 1/C_2$ |
| 初始电荷 | C1+A1 | $Q_0 = CV_0$ |
| 初始电流 | A1 | $I_0 = V_0 / R$ |
| 时间常数 | C1+A1 | $\tau = RC$ |
| 放电电流/电压 | C1+A1 | $x = x_0 e^{-t/\tau}$ |
| 储能 | C1+A1 | $W = \frac{1}{2} CV^2$ |
| 平均电流 (开关) | C1+C1+A1 | $I = fCV$ |

## 作图题模式

| 图形 | 标记 | 要点 |
|------|------|------|
| $I$-$t$ 放电 | B1+B1 | 指数衰减 + 从 $I_0$ 开始 + 渐近趋于 0 |
| $Q$-$t$ 放电 | B1+B1 | 指数衰减 + 从 $Q_0$ 开始 |
| $V$-$t$ 放电 | B1+B1 | 指数衰减 + 从 $V_0$ 开始 |
| $V$-$Q$ 图 | B1+B1 | 过原点的直线 + 面积 = 储能 |
| $I$-$t$ 充电 | B1+B1 | 指数衰减（从 $I_0$）到 0 |
| $V$-$t$ 充电 | B1+B1 | 指数上升（从 0 到 $V_0$） |

## 关键词速查

| 关键词 | 含义 |
|--------|------|
| charge per unit potential difference | 电容定义 |
| exponential decay | 指数衰减 |
| time constant $RC$ | 时间常数 |
| $x = x_0 e^{-t/RC}$ | 放电公式 |
| $W = \frac{1}{2} CV^2$ | 储能公式 |
| $fCV$ | 平均电流（开关电路） |
| potential–charge graph | $V$-$Q$ 图 |
| area under graph | 面积 = 储能 |
| smoothing | 平滑（电容在整流电路中的作用） |
| blocking d.c. | 隔直 |
