# Capacitance — 常见错误

### 1. 串并联电容公式和电阻搞混

- **错误**：电容串联用 $C = C_1 + C_2$
- **正确**：电容串联 $\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2}$，并联 $C = C_1 + C_2$（与电阻相反）
- **记忆**：电容串并联公式和电阻完全相反

### 2. RC 放电指数公式写错

- **错误**：放电用 $x = x_0 (1 - e^{-t/RC})$
- **正确**：放电 $x = x_0 e^{-t/RC}$，充电 $x = x_0 (1 - e^{-t/RC})$
- **检查**：$t = 0$ 时，放电公式 $x = x_0$（初始值）；充电公式 $x = 0$

### 3. 时间常数单位搞错

- **错误**：认为 $RC$ 的单位是 $\Omega$F（不直接）= s
- **正确**：$1\ \Omega \times 1\ \text{F} = 1\ \text{s}$
- **证明**：$\text{F} = \text{C/V} = \text{A·s/V}$，$\Omega \times \text{F} = (\text{V/A}) \times (\text{A·s/V}) = \text{s}$

### 4. 储能公式忘记 $\frac{1}{2}$

- **错误**：$W = QV$
- **正确**：$W = \frac{1}{2} QV = \frac{1}{2} CV^2$
- **原因**：充电过程中电压从 0 逐渐升高，平均电压为 $V/2$

### 5. $t = RC$ 时的数值记错

- **错误**：认为 $t = RC$ 时 $x/x_0 = 0.5$
- **正确**：$t = RC$ 时 $x/x_0 = e^{-1} \approx 0.37$
- **提示**：$t = RC \ln 2 \approx 0.693 RC$ 时 $x/x_0 = 0.5$

### 6. 初始电流计算错误

- **错误**：$I_0 = V_0 / R$ 被认为需要用到 $Q_0$ 计算
- **正确**：$t = 0$ 时，电容视为短路（完全充电状态），$I_0 = V_0 / R$
- **注意**：$R$ 是放电回路中的总电阻

### 7. 电容功能不熟悉

- **错误**：分不清电容器在电路中的功能
- **正确**：常见功能包括 smoothing（平滑）、timing（定时）、tuning（调谐）、blocking d.c.（隔直）、surge protection（浪涌保护）、temporary power supply（临时供电）
