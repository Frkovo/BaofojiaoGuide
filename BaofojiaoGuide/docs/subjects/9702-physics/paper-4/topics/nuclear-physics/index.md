# Nuclear Physics (Topic 23)

## 考纲要求
- $E = mc^2$ 质能等价
- mass defect 和 binding energy
- binding energy per nucleon 曲线
- nuclear fusion 和 nuclear fission
- 放射性衰变：random 和 spontaneous
- activity $A = \lambda N$，decay constant $\lambda$
- half-life $t_{1/2}$，$\lambda = 0.693 / t_{1/2}$
- 指数衰变 $x = x_0 e^{-\lambda t}$
- 核反应方程式

## 核心公式
| 公式 | 说明 |
|------|------|
| $E = \Delta m c^2$ | 质能等价 |
| $\Delta m = \sum m_{\text{反应前}} - \sum m_{\text{反应后}}$ | 质量亏损 |
| $A = \lambda N$ | activity 与 decay constant |
| $\lambda = 0.693 / t_{1/2}$ | decay constant 与 half-life |
| $x = x_0 e^{-\lambda t}$ | 指数衰变 |
| $1\text{ u} = 1.66 \times 10^{-27}\text{ kg}$ | 原子质量单位 |
| $c = 3.00 \times 10^8\text{ m s}^{-1}$ | 光速 |

## 常见错误
- 混淆 mass defect 和 binding energy
- 忘记将 u 转换为 kg
- 忽略 $E = \Delta m c^2$ 中的平方
- binding energy per nucleon 曲线形状记反
