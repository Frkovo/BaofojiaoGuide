# 考前速记 — Motion in a Circle

## Must-Know Formulas
| 公式 | 应用场景 |
|------|---------|
| $\theta = s/r$ | 弧度定义 |
| $\omega = \Delta\theta/\Delta t$ | 角速度 |
| $\omega = 2\pi/T = 2\pi f$ | 周期/频率转换 |
| $v = r\omega$ | 线-角速度 |
| $a = r\omega^2 = v^2/r$ | 向心加速度 |
| $F = mr\omega^2 = mv^2/r$ | 向心力 |

## 解题模式识别

| 问题模式 | 思路 |
|---------|------|
| 求向心力/加速度 | 直接用 $F = mv^2/r$ |
| 求周期/频率 | 先用 $\omega = 2\pi/T$ |
| 求线速度 | $v = r\omega$ |
| 竖直圆周最低点 | $T - mg = mv^2/r$ |
| 竖直圆周最高点 | $T + mg = mv^2/r$ |
| 卫星/行星轨道 | $GMm/r^2 = mv^2/r$（与13章结合） |

## 红色警报
- **向心力不是额外力** — 它是合力
- **$\omega$ 与 $f$ 差 $2\pi$**
- **弧度不是度**
- **cmd 到 m 转换**
- **最低点重力+向心力**
- **最高点重力-向心力**（当 $v$ 较小时可能掉下来）
