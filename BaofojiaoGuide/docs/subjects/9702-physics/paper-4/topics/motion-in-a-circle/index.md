# Motion in a Circle

## 考纲要求
- 定义弧度(radian)并用弧度表示角位移
- 理解并使用角速度(angular speed)的概念
- 应用 $\omega = 2\pi / T$ 和 $v = r\omega$
- 理解始终垂直于运动方向的恒定大小的力产生向心加速度(centripetal acceleration)
- 理解向心加速度导致恒定角速度的圆周运动
- 应用 $a = r\omega^2$ 和 $a = v^2 / r$
- 应用 $F = mr\omega^2$ 和 $F = mv^2 / r$

## 常见题型
- [题型分析](question-types.md)

## 核心公式
| 公式 | 说明 |
|------|------|
| $\theta = s / r$ | 弧度定义 |
| $\omega = \Delta\theta / \Delta t$ | 角速度定义 |
| $\omega = 2\pi / T$ | 角速度与周期 |
| $\omega = 2\pi f$ | 角速度与频率 |
| $v = r\omega$ | 线速度与角速度 |
| $a = r\omega^2 = v^2 / r$ | 向心加速度 |
| $F = mr\omega^2 = mv^2 / r$ | 向心力 |

## 常见错误
- 混淆角速度 $\omega$ 与频率 $f$：$\omega = 2\pi f$，不是 $\omega = f$
- 向心力不是独立的力，而是合力(resultant force)
- 忘记弧度: $\omega$ 单位是 rad s$^{-1}$
- 在竖直面圆周运动中，忘记考虑重力对向心力的贡献
