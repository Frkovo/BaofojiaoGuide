# Oscillations — 常见错误

### 1. 相位关系记错

- **错误**：位移最大时速度也最大
- **正确**：位移最大时速度为零（$v = \pm \omega \sqrt{x_0^2 - x^2}$，当 $x = x_0$ 时 $v = 0$）
- **记忆**：位移和速度相位差 $\pi/2$，位移和加速度反相

### 2. SHM 加速度方向记错

- **错误**：加速度与位移同向
- **正确**：$a = -\omega^2 x$，加速度始终指向平衡位置

### 3. 振幅和峰-峰值混淆

- **错误**：把图上的总行程（peak to peak）当作振幅
- **正确**：振幅 $x_0 =$ 总行程的一半（例如总行程 9.8 cm → $x_0 = 4.9$ cm）

### 4. 角频率 $\omega$ 和频率 $f$ 混淆

- **错误**：用 $f$ 代替 $\omega$ 代入 $a = -\omega^2 x$ 或 $E = \frac{1}{2} m \omega^2 x_0^2$
- **正确**：$\omega = 2\pi f$，$f$ 是每秒振动次数（Hz），$\omega$ 是角频率（rad s$^{-1}$）
- **检查**：$\omega$ 通常在 1–100 rad s$^{-1}$ 范围

### 5. $v$-$x$ 图解读错误

- **错误**：认为是直线或正弦
- **正确**：$v$-$x$ 图是**椭圆**（$v^2 = \omega^2 (x_0^2 - x^2)$）
- **提示**：从 $v$-$x$ 图的截距读 $v_0$ 和 $x_0$

### 6. 阻尼类型判断错误

- **错误**：认为 critical damping 最快停止（不回到平衡也算）
- **正确**：critical damping 最快回到平衡位置（rest position），不振荡
- **区分**：light damping 仍会振荡（振幅逐渐减小）；critical/heavy damping 不振荡

### 7. 共振条件记错

- **错误**：认为共振发生在驱动力最大时
- **正确**：共振发生在**驱动频率 = 系统自然频率**时
- **注意**：自然频率是系统自由振荡的频率（$f_0 = \frac{1}{2\pi} \sqrt{k/m}$ 或 $f_0 = \frac{1}{2\pi} \sqrt{g/L}$）

### 8. 能量转换频率错误

- **错误**：认为动能或势能的变化频率等于振动频率
- **正确**：每个振动周期内动能和势能各达到最大值两次，所以**能量变化频率是振动频率的 2 倍**
