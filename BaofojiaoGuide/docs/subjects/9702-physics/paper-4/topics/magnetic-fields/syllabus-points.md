# 考纲点 Syllabus Points

## 20.1 磁场概念 Concept of a Magnetic Field

- [ ] 理解磁场是由运动电荷或永磁体产生的力场
- [ ] 能用磁感线（field lines）表示磁场

## 20.2 载流导体受力 Force on a Current-Carrying Conductor

- [ ] 理解在磁场中的载流导体可能受力
- [ ] 掌握 $F = BIL\sin\theta$ 并会用弗莱明左手定则判断方向
- [ ] 定义磁通密度：$B = \frac{F}{IL}$，$B$ 是垂直通过单位电流单位长度导线的力

## 20.3 运动电荷受力 Force on a Moving Charge

- [ ] 判断在磁场中运动电荷的受力方向
- [ ] 掌握 $F = BQv\sin\theta$
- [ ] 理解 Hall 电压来源，推导和使用 $V_H = \frac{BI}{ntq}$（$t$ = 厚度）
- [ ] 理解 Hall probe 测量磁场的使用
- [ ] 描述带电粒子垂直射入均匀磁场的运动（圆周运动）
- [ ] 解释电场和磁场如何用于速度选择器（velocity selector）

## 20.4 电流产生的磁场 Magnetic Fields Due to Currents

- [ ] 画长直导线、圆形线圈和螺线管的磁场图
- [ ] 理解螺线管中加入铁芯会增强磁场
- [ ] 解释载流导体之间力的来源，判断方向

## 20.5 电磁感应 Electromagnetic Induction

- [ ] 定义磁通量：$\Phi = BA$（$B$ 垂直通过面积 $A$）
- [ ] 掌握 $\Phi = BA$ 和磁通链（magnetic flux linkage）$N\Phi$
- [ ] 理解并解释以下实验：
  - 变化的磁通在电路中感应出电动势
  - 感应电动势方向总是阻碍产生它的变化
  - 影响感应电动势大小的因素
- [ ] 掌握法拉第定律和楞次定律
