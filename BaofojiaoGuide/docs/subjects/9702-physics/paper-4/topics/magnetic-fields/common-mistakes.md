# 常见错误 Common Mistakes

1. **单位混淆 (cm vs m)**: 长度必须用米（m）计算。如 $d = 0.85$ cm 应写为 $0.85 \times 10^{-2}$ m

2. **Tesla 定义只写一半**: 必须写全 "newton per ampere per metre" 或 "force per unit current per unit length"。只写 "N A$^{-1}$ m$^{-1}$" 不加解释不算满分

3. **磁通定义忘记 normal/perpendicular**: $\Phi = BA$ 需要 $B$ **垂直**通过面积 $A$

4. **Faraday 定律漏掉 $N$**: 线圈有 $N$ 匝时，$E = N\frac{\Delta\Phi}{\Delta t}$，不要遗漏匝数

5. **Lenz's law 表述不准确**: 必须说 "opposes the **change** producing it"，不仅仅是 "opposes motion"

6. **Hall probe 中 $t$ 认错**: $t$ 是沿电流方向的厚度（如 PQ 或 PV），不是垂直于电流的宽度

7. **画磁场线方向错误**: 电流向纸内 → 顺时针，向纸外 → 逆时针（右手螺旋）

8. **平行导线力方向错误**: 记住 "like currents attract, opposite currents repel"

9. **解释涡电流只说一半**: 必须完整链条：变化磁场 → 感应电动势 → 涡电流 → 磁场相互作有力 → 阻碍运动

10. **磁通量图转电动势图**: 记住 $E \propto$ 斜率。水平线 → $E = 0$，斜线 → $E$ 常数
