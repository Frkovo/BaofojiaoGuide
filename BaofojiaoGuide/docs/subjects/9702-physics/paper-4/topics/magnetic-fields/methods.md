# 解题方法 Solution Methods

## 1. 求磁通密度 $B$ 的步骤

1. 写出 $F = BIL$ 或力矩平衡: 磁力矩 = 重力矩
2. 代入数据，注意单位转换（cm → m）
3. 解出 $B$

## 2. Faraday 定律计算

$$E = \frac{\Delta(N\Phi)}{\Delta t} = N\frac{\Delta(BA)}{\Delta t}$$

- 用 $B$-$t$ 图的斜率求 $\frac{\Delta B}{\Delta t}$
- $A$ 是线圈面积，用 $\pi r^2$（直径给半）
- $N$ 是匝数

## 3. Hall 电压计算

$$V_H = \frac{BI}{ntq}$$

- $n$: 电荷载流子数密度
- $t$: 通电方向上的厚度
- $q$: 载流子电荷量（通常 $e$）

## 4. 导线间磁力方向判断

- 同向电流 → 相互吸引
- 反向电流 → 相互排斥
- 用右手螺旋定则 + 弗莱明左手定则

## 5. 感应电动势方向的判断 (Lenz's law)

感应电流的方向总是阻碍磁通量的变化：
- 磁通增加 → 感应电流产生反向磁场
- 磁通减少 → 感应电流产生同向磁场

## 6. 平行导线问题

$$F = BIL$$

- $B$ 是另一根导线在该位置产生的磁通密度
- 两力大小相等（Newton's 3rd law）
