# 磁场与电磁感应 Magnetic Fields & Electromagnetic Induction

## 概述 Overview

Topic 20 涵盖**磁场（magnetic fields）** 和 **电磁感应（electromagnetic induction）** 。磁场由运动电荷或永磁体产生，对电流和运动电荷施加力。电磁感应描述变化的磁通量如何在导体中产生电动势。

:::note[Core Concepts]

- **Magnetic flux density $B$** (磁通密度/磁感应强度): 单位 Tesla (T)
- **Force on a current-carrying conductor**: $F = BIL\sin\theta$
- **Force on a moving charge**: $F = BQv\sin\theta$
- **Hall effect**: $V_H = \frac{BI}{ntq}$
- **Magnetic flux**: $\Phi = BA$
- **Faraday's law**: $E = -\frac{d(N\Phi)}{dt}$
- **Lenz's law**: 感应电动势的方向总是阻碍产生它的变化

:::

## 考试占比 Exam Weighting

磁场 + 电磁感应通常在 Paper 4 中占 **10–15 分**，约 10–15% 总分。常以单独大题出现（如 Q8, Q9），结合力学平衡或电路分析。
