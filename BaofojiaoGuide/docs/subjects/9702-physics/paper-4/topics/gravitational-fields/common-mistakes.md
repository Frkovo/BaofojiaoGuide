# 常见错误 — Gravitational Fields

## 1. 混淆 $g$ 和 $G$

**错误**: 用 $g = 9.81$ 代表引力常数。

**正确**: $G = 6.67 \times 10^{-11}$ N m$^2$ kg$^{-2}$ 是**引力常数**，$g = 9.81$ N kg$^{-1}$ 是地球表面**引力场强度**。

**Fix**: $G$ 是宇宙常数，$g$ 取决于位置。

## 2. 引力势和势能的正负号

**错误**: 说 gravitational potential 是正的，或者 $\phi = +GM/r$。

**正确**: $\phi = -GM/r$，引力势是负的，在无穷远处为零。

**Fix**: 引力是吸引力，从无穷远移过来做功为正，所以势能减少为负。

## 3. $r$ 从表面算起

**错误**: 地球表面高度 $h$ 处用 $g = GM/h^2$。

**正确**: $r = R + h$（从地心算起），$g = GM/(R+h)^2$。

**Fix**: 所有引力公式中的 $r$ 都是距离**中心**的距离。

## 4. 地球表面 $g$ 恒定误解

**错误**: 认为 $g = GM/R^2$ 在任何高度都成立。

**正确**: 只有小高度变化（$h \ll R$）时 $g$ 才近似常数。

**Fix**: 当 $h$ 较大时必须用 $g = GM/(R+h)^2$。

## 5. 卫星轨道的周期关系

**错误**: 认为周期与半径成正比。

**正确**: $T^2 \propto r^3$（开普勒第三定律），半径增加，周期增加更快。

**Fix**: 用 $T^2 = (4\pi^2/GM)r^3$ 推导。

## 6. 同步轨道条件遗漏

**错误**: 只说周期24小时就算 geostationary。

**正确**: 还需要：在赤道正上方，从西向东运动。

**Fix**: 三个条件缺一不可：24h周期、赤道上空、西向东。
