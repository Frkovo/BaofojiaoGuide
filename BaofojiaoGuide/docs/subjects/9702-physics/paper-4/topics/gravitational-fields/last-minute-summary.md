# 考前速记 — Gravitational Fields

## Must-Know Formulas
| 公式 | 说明 |
|------|------|
| $F = Gm_1m_2/r^2$ | 万有引力 |
| $g = F/m$ | 场强定义 |
| $g = GM/r^2$ | 点质量场强 |
| $\phi = -GM/r$ | 引力势 |
| $E_P = -GMm/r$ | 引力势能 |
| $T^2 = (4\pi^2/GM)r^3$ | 轨道周期 |

## 解题模式识别

| 问题模式 | 思路 |
|---------|------|
| 求表面引力场强 | $g = GM/R^2$ |
| 求某高度引力场强 | $g = GM/(R+h)^2$ |
| 卫星/行星轨道 | $GM/r^2 = v^2/r$ |
| 求势能变化 | $\Delta E_P = GMm(1/r_1 - 1/r_2)$ |
| 逃逸速度 | $v = \sqrt{2GM/r}$ |
| 同步卫星周期 | $T = 24$ h |

## 红色警报
- **$G$ 不等于 $g$**
- **$r$ 从中心算起**
- **势能为负**
- **$M$ 是中心天体质量**
- **卫星质量消去**
- **同步卫星三条件**

## Quick Checklist
- [ ] 区分 $G$ (constant) 和 $g$ (field strength)
- [ ] $r$ 从中心算起
- [ ] 势能公式负号
- [ ] 轨道方程正确
- [ ] 单位：$g$ in N kg$^{-1}$ or m s$^{-2}$
