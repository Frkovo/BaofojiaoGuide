# Gravitational Fields

## 考纲要求
- 理解引力场是力场，定义引力场强度为力每单位质量
- 用场线表示引力场
- 理解均匀球体外一点，质量可视为集中于球心
- 应用牛顿万有引力定律 $F = Gm_1m_2/r^2$
- 分析引力提供向心力的圆形轨道
- 理解地球同步轨道(geostationary orbit)的特点
- 推导 $g = GM/r^2$
- 应用 $g = GM/r^2$
- 理解地球表面附近 $g$ 近似为常数
- 定义引力势(gravitational potential)
- 应用 $\phi = -GM/r$
- 应用 $E_P = -GMm/r$

## 常见题型
- [题型分析](question-types.md)

## 核心公式
| 公式 | 说明 |
|------|------|
| $F = Gm_1m_2/r^2$ | 万有引力定律 |
| $g = F/m$ | 引力场强度定义 |
| $g = GM/r^2$ | 点质量引力场强度 |
| $\phi = -GM/r$ | 引力势 |
| $E_P = -GMm/r$ | 引力势能 |
| $GMm/r^2 = mv^2/r$ | 轨道运动方程 |

## 常见错误
- 引力势为负值，越靠近中心越负
- 混淆 $g$（场强）和 $G$（引力常数）
- 地球表面近似 $g$ constant 只适用于小高度变化
- 忘记 $r$ 从**中心**算起，不是表面
