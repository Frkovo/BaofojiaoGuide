# 常见错误 Common Mistakes

1. **单位混乱**: eV 和 J 的互换。$1$ eV $= 1.60 \times 10^{-19}$ J。题目给 J 就用 J，给 eV 就用 eV

2. **功率与频率混淆**: 功率 $P$ 决定光子数（每秒光子数 = $P/hf$），频率 $f$ 决定每个光子的能量。增加频率不增加功率

3. **$hf = \Phi + \frac12 mv^2$ 中忘记 $E_K$ 是最大值**: 光电子有不同动能，方程只给出最大动能

4. **截止频率定义不准确**: 必须说 "minimum frequency for photoemission to occur"，不能只说 "frequency at which photoelectrons are emitted"

5. **光子动量方向**: 电子在与光子碰撞中获动量，光子动量减少（波长增大），不可能减少波长

6. **光子动量公式记错**: $p = E/c = hf/c = h/\lambda$，不是 $p = hf$

7. **E vs f 图画错**: 正确图线是直线从 $f_0$ 开始上升，斜率为 $h$。不是曲线

8. **电子衍射解释不够完整**: 不能只说 "electrons behave as waves"，必须说 "diffraction pattern shows wave nature"

9. **能级跃迁方向颠倒**: 吸收光子 → 电子 Up（$E_{\text{higher}} - E_{\text{lower}}$ 正），发射光子 → 电子 Down（差值负）

10. **吸收光谱解释不准**: 原子吸收特定能量的光子后跃迁到高能级，再向各方向发射 → 我们看到的缺失线是吸收后重发射的方向随机造成
