# 量子物理 Quantum Physics

## 概述 Overview

Topic 22 涵盖光的粒子性、光电效应、波粒二象性和原子能级光谱。这是 Paper 4 中概念性最强的 topic 之一，需要理解实验证据和理论推导。

:::note[Core Concepts]

- **Photon energy**: $E = hf = \frac{hc}{\lambda}$
- **Photoelectric effect**: $hf = \Phi + \frac12 mv_{\text{max}}^2$
- **Threshold frequency**: $f_0 = \frac{\Phi}{h}$ (最小频率才能释放光电子)
- **de Broglie wavelength**: $\lambda = \frac{h}{p}$
- **Energy levels**: $hf = E_1 - E_2$ (离散能级跃迁)
- **Electronvolt (eV)**: $1$ eV $= 1.60 \times 10^{-19}$ J

:::
