# 考纲点 Syllabus Points

## 22.1 光子的能量和动量 Energy and Momentum of a Photon

- [ ] 理解电磁辐射具有粒子性（particulate nature）
- [ ] 理解光子是电磁能量的量子（quantum）
- [ ] 掌握 $E = hf$
- [ ] 使用电子伏特（eV）作为能量单位
- [ ] 理解光子有动量 $p = E/c$

## 22.2 光电效应 Photoelectric Effect

- [ ] 理解照射金属表面的电磁辐射可能释放光电子（photoelectrons）
- [ ] 理解和运用截止频率（threshold frequency）和截止波长（threshold wavelength）
- [ ] 用光子能量和功函数（work function energy）解释光电发射
- [ ] 掌握 $hf = \Phi + \frac12 mv_{\text{max}}^2$
- [ ] 解释为什么光电子的最大动能与光强无关，而光电流与光强成正比

## 22.3 波粒二象性 Wave-Particle Duality

- [ ] 理解光电效应是电磁辐射粒子性的证据，干涉和衍射是波动性的证据
- [ ] 定性描述和解释电子衍射实验对粒子波动性的证据
- [ ] 理解 de Broglie 波长是运动粒子的伴随波长
- [ ] 掌握 $\lambda = h/p$

## 22.4 原子能级与线光谱 Energy Levels in Atoms and Line Spectra

- [ ] 理解孤立原子中有离散的电子能级（如氢原子）
- [ ] 理解发射光谱（emission spectra）和吸收光谱（absorption line spectra）的形成
- [ ] 掌握 $hf = E_1 - E_2$
