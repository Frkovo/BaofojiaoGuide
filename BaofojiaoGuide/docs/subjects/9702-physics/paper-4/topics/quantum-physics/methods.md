# 解题方法 Solution Methods

## 1. 光子能量计算

$$E = hf = \frac{hc}{\lambda}$$

- 注意 $\lambda$ 必须以米（m）为单位
- nm → m: $\times 10^{-9}$
- eV → J: $\times 1.60 \times 10^{-19}$

## 2. 光电效应方程

$$hf = \Phi + E_{K,\text{max}}$$

- $\Phi$: work function energy (功函数)
- $E_{K,\text{max}} = \frac12 mv_{\text{max}}^2$
- 截止频率: $f_0 = \Phi / h$
- 截止波长: $\lambda_0 = hc / \Phi$

## 3. 光强与光子数

$$P = n \times hf$$

- $P$: 辐射功率
- $n$: 每秒光子数
- 频率增大而功率不变 → 光子数减少

## 4. de Broglie 波长

$$\lambda = \frac{h}{p} = \frac{h}{mv}$$

## 5. 能级跃迁

$$hf = E_{\text{higher}} - E_{\text{lower}}$$

- 发射光谱: 电子从高能级→低能级，释放光子
- 吸收光谱: 电子从低能级→高能级，吸收光子

## 6. 正负电子湮灭

$$E = 2m_e c^2 \quad (\text{两个光子总能量})$$

每个光子能量 $= m_e c^2$ (因为动量守恒，两光子反向射出)
