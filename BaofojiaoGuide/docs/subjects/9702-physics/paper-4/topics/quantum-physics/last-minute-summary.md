# 考前速记 Last Minute Summary

## 核心公式 Key Formulas

$$E = hf = \frac{hc}{\lambda}$$
$$p = \frac{E}{c} = \frac{h}{\lambda}$$
$$hf = \Phi + \frac12 mv_{\text{max}}^2 \quad \text{(Photoelectric effect)}$$
$$\lambda = \frac{h}{p} \quad \text{(de Broglie)}$$
$$hf = E_1 - E_2 \quad \text{(Energy levels)}$$

## 常数 Memorise

- $h = 6.63 \times 10^{-34}$ J s
- $e = 1.60 \times 10^{-19}$ C
- $1$ eV $= 1.60 \times 10^{-19}$ J
- $c = 3.00 \times 10^8$ m s$^{-1}$
- $m_e = 9.11 \times 10^{-31}$ kg
