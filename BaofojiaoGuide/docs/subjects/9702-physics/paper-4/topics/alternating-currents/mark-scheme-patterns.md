# 评分模式 Mark Scheme Patterns

## 计算题的标准评分结构

| 步骤 | 标记 | 说明 |
|------|------|------|
| 写出 $V_0 = V_{\text{r.m.s.}} \times \sqrt{2}$ | **B1** 或 **C1** | 可能直接使用公式 |
| 代入并算出 $V_0$ | **B1** | 有时被隐含 |
| 代入 $P = V_0^2/R$ | **C1** | 后续计算的基础 |
| 最终答案 | **A1** | 含单位 |

## 画图题的评分特点

- **功率-时间图**: 必须有正确的波峰数值、形状、周期数
- **B1** peak at correct value
- **B1** correct sinusoidal shape
- **B1** correct number of cycles / period

## 解释题的关键词

- **半波整流**: "one direction is removed"
- **全波整流**: "one direction is reversed"
- **平滑**: "capacitor discharges through resistor"
