# 考纲点 Syllabus Points

## 21.1 交流电特征 Characteristics of Alternating Currents

- [ ] 理解交流电/电压的周期（period）、频率（frequency）和峰值（peak value）
- [ ] 使用 $x = x_0 \sin\omega t$ 表示正弦交流
- [ ] 知道正弦交流电阻性负载的平均功率等于最大功率的一半
- [ ] 区分 r.m.s. 和峰值，掌握 $I_{\text{r.m.s.}} = I_0/\sqrt{2}$, $V_{\text{r.m.s.}} = V_0/\sqrt{2}$

## 21.2 整流与滤波 Rectification and Smoothing

- [ ] 从图形区分半波整流（half-wave）和全波整流（full-wave）
- [ ] 解释单个二极管用于半波整流
- [ ] 解释四个二极管（桥式整流器）用于全波整流
- [ ] 分析单个电容的平滑效果，包括电容值和负载电阻的影响
