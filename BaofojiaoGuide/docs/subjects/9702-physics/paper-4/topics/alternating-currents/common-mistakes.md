# 常见错误 Common Mistakes

1. **混淆 peak 和 r.m.s.**: $V_0 = V_{\text{r.m.s.}} \times \sqrt{2}$ 而不是除以 $\sqrt{2}$

2. **平均功率 = 一半最大功率**只适用于纯正弦交流电阻性负载，不适用于整流后的波形

3. **二极管方向画反**: 二极管箭头方向是允许电流方向（从正到负）

4. **桥式整流器中二极管排列错误**: 四个二极管必须形成桥式结构

5. **电容平滑解释不完整**: 必须说 "capacitor charges to peak voltage, then discharges through $R$ when voltage falls"

6. **$RC$ 时间常数影响**: 更大的 $R$ 或 $C$ → 更长的放电时间 → 更好的平滑，但 $RC$ 太大会导致跟不上输入变化

7. **功率图中忘记示波器基线**: 正弦功率波应该全部在正半轴（$P \geq 0$），因为功率始终为正值

8. **频率单位疏忽**: 注意 kHz 与 Hz 的换算（$50$ kHz $= 50 \times 10^3$ Hz）
