# 交流电 Alternating Currents

## 概述 Overview

Topic 21 涵盖正弦交流电的特征、整流（rectification）和滤波（smoothing）。重点在于 r.m.s. 值、平均功率、半波/全波整流以及电容平滑。

:::note[Core Concepts]

- **Alternating current/voltage**: $x = x_0 \sin\omega t$
- **r.m.s. values**: $I_{\text{r.m.s.}} = \frac{I_0}{\sqrt{2}}$, $V_{\text{r.m.s.}} = \frac{V_0}{\sqrt{2}}$
- **Mean power**: $\langle P \rangle = \frac{1}{2}P_{\text{max}}$ for sinusoidal
- **Rectification**: half-wave (single diode) vs full-wave (bridge rectifier)
- **Smoothing**: capacitor discharges through load resistor between peaks

:::
