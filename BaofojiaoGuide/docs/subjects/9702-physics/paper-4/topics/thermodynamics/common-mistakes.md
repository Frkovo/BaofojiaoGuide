# Thermodynamics — 常见错误

### 1. 热力学第一定律符号搞反

- **错误**：系统对外做功时 $W$ 为正
- **正确**：$\Delta U = q + W$，系统对外做功时 $W$ 为负
- **提示**：气体膨胀 → $W$ 为负 → $\Delta U = q - p\Delta V$

### 2. "Work done on gas" 与 "Work done by gas" 混淆

- **错误**：把题目给的 "work done by the gas" 直接当作 $+W$ 代入
- **正确**："work done by gas" = 气体对外做功 = $-$值；"work done on gas" = 外界对气体做功 = $+$值
- **提示**：代入公式前统一转换为 $\Delta U = q + W$（$W$ 是 work done **on** gas）

### 3. $p$-$V$ 图面积正负判断错误

- **错误**：只看面积数值，忽略过程方向
- **正确**：$V$ 增大 → $W$ 负（气体对外）；$V$ 减小 → $W$ 正（外界对气体）
- **提示**：顺时针循环 → 净功为正（热机）；逆时针 → 净功为负（热泵）

### 4. 混淆温度变化与内能变化的关系

- **错误**：认为温度不变时 $\Delta U$ 也为零（对固体/液体/真实气体）
- **正确**：$\Delta U = 0$ 仅对**理想气体**成立（因为无分子势能变化）；实际物质的相变（熔化/沸腾）时温度不变但 $\Delta U \neq 0$

### 5. 等容（isochoric）与等压（isobaric）搞混

- **错误**：等容过程认为 $q = 0$
- **正确**：等容（$V$ 不变）→ $W = 0$ → $\Delta U = q$；绝热（$q = 0$）→ $\Delta U = W$
- **区分**：等容看体积，绝热看热交换

### 6. 忘记 $\Delta U$ 是状态函数

- **错误**：在表格题中分别计算每行的 $\Delta U$，不检查总和
- **正确**：循环过程中各行的 $\Delta U$ 之和必须为零（初末状态相同）
- **提示**：最后一行的 $\Delta U$ 可以通过前几行之和来验证
