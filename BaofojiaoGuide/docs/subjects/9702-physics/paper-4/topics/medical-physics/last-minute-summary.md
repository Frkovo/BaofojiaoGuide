# Last Minute Summary — Medical Physics

## 必记公式

| 公式 | 适用场景 |
|------|----------|
| $Z = \rho c$ | 特定声阻抗 |
| $\dfrac{I_R}{I_0} = \dfrac{(Z_2 - Z_1)^2}{(Z_2 + Z_1)^2}$ | 反射系数 |
| $I = I_0 e^{-\mu x}$ | 衰减（ultrasound 和 X-ray） |
| $\lambda_{\min} = hc / eV$ | X-ray 最短波长 |
| $E_{\gamma} = m_e c^2 = 8.2 \times 10^{-14}\text{ J}$ | 湮灭光子能量 |

## 常数值
- $m_e = 9.11 \times 10^{-31}\text{ kg}$
- $c = 3.00 \times 10^8\text{ m s}^{-1}$
- $h = 6.63 \times 10^{-34}\text{ J s}$
- $e = 1.60 \times 10^{-19}\text{ C}$
- 人体组织 $Z \approx 1.6 \times 10^6\text{ kg m}^{-2}\text{ s}^{-1}$

## 考前检查清单

### Ultrasound
- [ ] Piezoelectric 效应：电压→形变，形变→电压
- [ ] 频率范围：> 20 kHz（超声波）
- [ ] 反射回波时间→深度信息
- [ ] $Z = \rho c$ 计算
- [ ] 反射系数公式
- [ ] Gel 的作用：匹配 acoustic impedance

### X-rays
- [ ] 产生：高速电子撞击金属靶
- [ ] 硬度 = 电压控制，强度 = 电流控制
- [ ] $\lambda_{\min} = hc / eV$
- [ ] $I = I_0 e^{-\mu x}$
- [ ] Contrast：不同组织吸收差异
- [ ] CT：多角度扫描 → 2D 截面 → 3D

### PET scanning
- [ ] Tracer：含 $\beta^+$ 核素的物质
- [ ] $\beta^+$ decay → positron
- [ ] Annihilation: $e^+ + e^- \rightarrow 2\gamma$
- [ ] 每个 photon 能量 $= m_e c^2$
- [ ] 方向相反（动量守恒）
- [ ] 探测符合事件 → 定位 → 成像
