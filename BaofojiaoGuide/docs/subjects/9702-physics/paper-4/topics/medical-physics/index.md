# Medical Physics (Topic 24)

## 考纲要求
- piezoelectric transducer 产生和探测 ultrasound
- specific acoustic impedance $Z = \rho c$
- intensity reflection coefficient $I_R / I_0 = (Z_2 - Z_1)^2 / (Z_2 + Z_1)^2$
- attenuation $I = I_0 e^{-\mu x}$（ultrasound 和 X-rays）
- X-ray 产生：电子轰击金属靶
- CT scanning：3D image 原理
- PET scanning：$\beta^+$ decay、annihilation、gamma-ray photons
- tracer 概念

## 核心公式
| 公式 | 说明 |
|------|------|
| $Z = \rho c$ | 特定声阻抗 |
| $\dfrac{I_R}{I_0} = \dfrac{(Z_2 - Z_1)^2}{(Z_2 + Z_1)^2}$ | 强度反射系数 |
| $I = I_0 e^{-\mu x}$ | 衰减公式 |
| $\lambda_{\min} = hc / eV$ | X-ray 最短波长 |
| $E = mc^2$ | 湮灭能量 |

## 常见错误
- 混淆 ultrasound 和 X-rays 的产生方式
- 忘记 gel 的作用是减少反射
- CT 和 PET 概念混淆
- attenuation 公式中 $\mu$ 单位（cm$^{-1}$ 或 m$^{-1}$）
