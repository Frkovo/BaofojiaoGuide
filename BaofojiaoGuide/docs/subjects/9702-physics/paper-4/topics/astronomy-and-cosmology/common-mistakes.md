# Common Mistakes — Astronomy and Cosmology

## Mistake 1: 混淆 luminosity 和 radiant flux

**错误**：认为 luminosity 和 radiant flux 是同一概念。

**正确**：
- **Luminosity $L$**: 恒星的总辐射功率（单位 W），是恒星本身的属性
- **Radiant flux $F$**: 在地球接收到的单位面积功率（单位 W m$^{-2}$），与距离有关

## Mistake 2: 忘记距离平方反比公式中的 $4\pi$

**错误**：写成 $F = L / d^2$。

**正确**：$F = L / (4\pi d^2)$，因为能量均匀分布在球面上。

## Mistake 3: Wien's law 中温度和波长的关系搞反

**错误**：认为 $\lambda_{\max} \propto T$（温度越高波长越长）。

**正确**：$\lambda_{\max} \propto 1 / T$，温度越高，峰值波长越短（蓝移）。

## Mistake 4: Stefan-Boltzmann 公式中忘记 $r^2$

**错误**：写成 $L = 4\pi\sigma T^4$ 或 $L = 4\pi\sigma r T^4$。

**正确**：$L = 4\pi\sigma r^2 T^4$，与半径平方成正比。

## Mistake 5: Redshift 公式符号错误

**错误**：认为红移意味着 $\lambda_{\text{obs}} < \lambda_{\text{emit}}$。

**正确**：Redshift 是 $\lambda_{\text{obs}} > \lambda_{\text{emit}}$（波长变长），光源正在远离。

## Mistake 6: Hubble 公式中的单位

**错误**：使用 km s$^{-1}$ Mpc$^{-1}$ 与 SI 单位混淆。

**正确**：在 9702 考试中，使用 SI 单位，$H_0$ 单位是 s$^{-1}$。

## Mistake 7: 混淆 standard candle 和普通恒星

**错误**：认为任何恒星都可以作为 standard candle。

**正确**：Standard candle 是 luminosity 已知的天体（如特定类型的超新星或变星），不是所有恒星。

## Mistake 8: 宇宙年龄公式记错

**错误**：认为宇宙年龄 $= H_0$。

**正确**：宇宙年龄 $\approx 1 / H_0$（假设匀速膨胀）。
