# Mark Scheme Patterns — Astronomy and Cosmology

## 常见标记类型

### B1 (独立分)
- 定义 luminosity
- 定义 standard candle
- 定义 redshift
- 解释宇宙膨胀的证据
- 解释 Big Bang theory
- 识别 Hubble constant

### C1 (补偿分)
- 代入 $F = L / (4\pi d^2)$
- 代入 $L = 4\pi\sigma r^2 T^4$
- 代入 $\Delta\lambda / \lambda = v / c$
- 代入 $v = H_0 d$

### A1 (答案分)
- 数值答案 + 单位
- 正确计算 $H_0$

## 典型评分模式

### Luminosity / flux 题
- **B1**: 定义 luminosity
- **C1**: 代入 $F = L / (4\pi d^2)$
- **A1**: 数值 + 单位

### Stellar radius 题
- **C1**: 代入 Stefan-Boltzmann 公式
- **A1**: 半径数值

### Redshift + Hubble 题
- **B1/B1**: redshift 定义 + 宇宙膨胀
- **C1**: $\Delta\lambda / \lambda = v / c$
- **C1**: $v = H_0 d$
- **A1**: $H_0$ 数值

## 注意事项
- Hubble constant 单位必须是 s$^{-1}$（SI）
- Luminosity 单位 W，flux 单位 W m$^{-2}$
- 距离必须用米（m），SI 单位
- Stefan-Boltzmann 常数 $\sigma = 5.67 \times 10^{-8}\text{ W m}^{-2}\text{ K}^{-4}$ 在 data booklet 中
- Redshift 公式使用约等于号
