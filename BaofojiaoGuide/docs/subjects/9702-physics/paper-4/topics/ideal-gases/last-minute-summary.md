# 考前速记 — Ideal Gases

## Must-Know Formulas
| 公式 | 说明 |
|------|------|
| $pV = nRT$ | 状态方程（摩尔） |
| $pV = NkT$ | 状态方程（分子） |
| $k = R/N_A$ | 玻尔兹曼常数 |
| $pV = \frac13 Nm\langle c^2\rangle$ | 气体动理论 |
| $\frac12 m\langle c^2\rangle = \frac32 kT$ | 分子平均动能 |
| $U = \frac32 NkT = \frac32 nRT$ | 单原子气体内能 |
| $c_{\text{r.m.s.}} = \sqrt{3kT/m}$ | 均方根速度 |

## 解题模式识别

| 问题模式 | 思路 |
|---------|------|
| 求分子数 | $N = pV/(kT)$ |
| 求摩尔数 | $n = pV/(RT)$ |
| 求 r.m.s. speed | $c_{\text{r.m.s.}} = \sqrt{3kT/m}$ |
| 求平均动能 | $\langle E_K \rangle = \frac32 kT$ |
| 推导分子动能 | 联立 $pV = \frac13 Nm\langle c^2\rangle$ 和 $pV = NkT$ |
| 陈述假设 | 4条：随机运动、弹性碰撞、无作用力、体积可忽略 |

## 红色警报
- **温度用 K，不能用 °C**
- **$n$ 是摩尔数，$N$ 是分子数**
- **$\langle c^2\rangle$ ≠ $(c_{\text{mean}})^2$**
- **理想气体无势能**
- **$m$ 是单分子质量**
- **$R = 8.31$ J mol$^{-1}$ K$^{-1}$**
