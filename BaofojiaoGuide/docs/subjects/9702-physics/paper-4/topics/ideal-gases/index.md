# Ideal Gases

## 考纲要求
- 理解物质的量(amount of substance)是 SI 基本量，单位 mol
- 使用摩尔量：1 mol 包含阿伏伽德罗常数 $N_A$ 个粒子
- 理解服从 $pV \propto T$ 的气体称为理想气体(ideal gas)
- 应用理想气体状态方程 $pV = nRT$ 和 $pV = NkT$
- 玻尔兹曼常数 $k = R/N_A$
- 陈述气体动理论的基本假设
- 用分子运动解释气压，推导 $pV = \frac13 Nm\langle c^2\rangle$
- 理解均方根速度 $c_{\text{r.m.s.}} = \sqrt{\langle c^2\rangle}$
- 比较 $pV = \frac13 Nm\langle c^2\rangle$ 与 $pV = NkT$ 得出分子平均平动动能 $\frac32 kT$

## 核心公式
| 公式 | 说明 |
|------|------|
| $pV = nRT$ | 状态方程（摩尔形式） |
| $pV = NkT$ | 状态方程（分子数形式） |
| $k = R/N_A$ | 玻尔兹曼常数 |
| $pV = \frac13 Nm\langle c^2\rangle$ | 气体动理论方程 |
| $\frac12 m\langle c^2\rangle = \frac32 kT$ | 分子平均动能 |
| $c_{\text{r.m.s.}} = \sqrt{\langle c^2\rangle}$ | 均方根速度 |

## 常见错误
- 温度必须用开尔文(K)，不是摄氏度(°C)
- 区分 $n$ (摩尔数) 和 $N$ (分子数)：$N = nN_A$
- $\langle c^2\rangle$ 是 mean-square speed，不是 mean speed 的平方
- $c_{\text{r.m.s.}}$ 不等于平均速度
