# Temperature

## 考纲要求
- 理解热能从高温区域传向低温区域
- 理解等温区域处于热平衡(thermal equilibrium)
- 理解随温度变化的物理性质可用于测温，举例（液体密度、恒压气体体积、金属电阻、热电偶电动势）
- 理解热力学温标(thermodynamic temperature)不依赖特定物质的性质
- 开尔文与摄氏度换算：$T/K = \theta/°C + 273.15$
- 理解最低可能温度为零开尔文（绝对零度 absolute zero）
- 定义并使用比热容(specific heat capacity)
- 定义并使用比潜热(specific latent heat)，区分熔解潜热和汽化潜热

## 核心公式
| 公式 | 说明 |
|------|------|
| $T/K = \theta/°C + 273.15$ | 温度换算 |
| $Q = mc\Delta\theta$ | 比热容 |
| $Q = mL$ | 比潜热 |
| $\Delta E_P = mc\Delta\theta$ （升温） | 热能变化 |
| $\Delta E_P = mL$ （相变） | 潜热 |

## 常见错误
- 温度变化 $\Delta T$ 在两种温标中数值相同（$\Delta T$ in K = $\Delta\theta$ in °C）
- 比热容和比潜热是不同的概念
- 混淆 specific latent heat of fusion 和 vaporisation
