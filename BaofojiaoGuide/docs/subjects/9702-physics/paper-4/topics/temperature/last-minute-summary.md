# 考前速记 — Temperature

## Must-Know Formulas
| 公式 | 说明 |
|------|------|
| $T/K = \theta/°C + 273.15$ | 温度换算 |
| $Q = mc\Delta\theta$ | 比热容 |
| $Q = mL$ | 比潜热 |
| $\Delta T \text{ (K)} = \Delta\theta \text{ (°C)}$ | 温差等价 |

## 解题模式识别

| 问题模式 | 思路 |
|---------|------|
| 求比热容 | $c = Q/(m\Delta\theta)$ |
| 求比潜热 | $L = Q/m$ |
| 混合温度 | $m_1c_1(\theta_1 - \theta_f) = m_2c_2(\theta_f - \theta_2)$ |
| 包含相变 | 分步：升温 + 相变 |
| 测温性质 | 密度、电阻、电动势、恒压气体体积 |

## 红色警报
- **相变温度不变**
- **K 和 °C 的温差相同**
- **$Q = mL$ 是相变，$Q = mc\Delta\theta$ 是升温**
- **热力学公式中用 K**
- **定义题：比热容 = energy per unit mass per unit temp change**
- **比潜热 = energy per unit mass (during change of state)**
