# MS 关键词速查

## 通用评分术语

| MS 原文 | 中文含义 | 说明 |
|---------|---------|------|
| ecf (error carried forward) | 错误向前传递 | 前一步算错，后一步用错数但方法正确，后一步仍给分 |
| AE (arithmetic error) | 算术错误 | 计算过程正确但最终结果算错，扣答案分 |
| POT (power of ten) | 10 的幂次错误 | 如 $10^3$ 写成 $10^4$，扣 1 分 |
| SF (significant figures) | 有效数字 | 答案有效数字不符合要求时扣 1 分 |
| XP | 物理公式错误 | 使用了错误的物理公式，后续步骤不给分 |
| BOD (benefit of doubt) | 疑点利益归考生 | 模棱两可的表述视为正确 |
| ora (or reverse argument) | 或反向论证 | 从相反方向论证同样给分 |
| ( ) | 括号内容 | 不需要写但写了帮助理解 |
