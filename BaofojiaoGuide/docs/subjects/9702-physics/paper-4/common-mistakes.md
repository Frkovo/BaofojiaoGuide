# 常见错误

## 12 Motion in a Circle

### 1. 混淆角速度和线速度
- **错误**：直接用 $v$ 计算向心力而没先转换 $\omega$
- **正确**：$v = r\omega$，然后 $F = mr\omega^2$ 或 $F = mv^2/r$
- **提示**：题目给的是转速 $n$（revolutions per second）时，$\omega = 2\pi n$

### 2. 向心加速度方向画错
- **错误**：认为向心加速度沿切线方向
- **正确**：向心加速度始终指向圆心

## 13 Gravitational Fields

### 1. 混淆引力场强度和引力势
- **错误**：$g$ 和 $\phi$ 都认为是标量
- **正确**：$g$ 是矢量（有方向），$\phi$ 是标量；$g = -\frac{d\phi}{dr}$（在地球表面附近 $g$ 近似常数）
- **注意**：引力势 $\phi$ 为负值，越接近天体负得越多

### 2. 引力势能符号错误
- **错误**：$E_p = \frac{GMm}{r}$（正号）
- **正确**：$E_p = -\frac{GMm}{r}$（负号，表示束缚态）
- **检查**：卫星远离地球时 $E_p$ 从负值趋于零

## 14/15 Temperature & Ideal Gases

### 1. 温度单位不转换
- **错误**：摄氏温度直接代入 $pV = nRT$
- **正确**：必须用开尔文 $T(K) = \theta(^\circ C) + 273.15$
- **提示**：$\Delta T$ 在摄氏和开尔文下数值相同

### 2. 混淆 $n$ 和 $N$
- **错误**：$pV = NkT$ 中的 $N$ 写成摩尔数
- **正确**：$n$ 是摩尔数，$N = nN_A$ 是分子总数
- **区分**：$pV = nRT$（$R$ 是摩尔气体常数）vs $pV = NkT$（$k$ 是玻尔兹曼常数）

### 3. r.m.s. 速度公式符号
- **错误**：写 $\langle c^2 \rangle = \langle c \rangle^2$
- **正确**：$\langle c^2 \rangle \neq \langle c \rangle^2$，r.m.s. = $\sqrt{\langle c^2 \rangle}$
- **公式**：$\frac12 m \langle c^2 \rangle = \frac32 kT$

## 16 Thermodynamics

### 1. 热力学第一定律符号搞反
- **错误**：系统对外做功时 $W$ 为正
- **正确**：$\Delta U = q + W$，系统对外做功时 $W$ 为负（$W = -p\Delta V$ 当 $\Delta V > 0$）
- **提示**：气体膨胀 → $W$ 为负 → $\Delta U = q - p\Delta V$

## 17 Oscillations

### 1. 相位关系记错
- **错误**：位移最大时速度也最大
- **正确**：位移最大时速度为零（$v = \pm \omega\sqrt{x_0^2 - x^2}$，$x = x_0$ 时 $v = 0$）
- **记忆**：位移和速度相位差 $\pi/2$，位移和加速度同相（反相）

### 2. SHM 加速度方向记错
- **错误**：加速度与位移同向
- **正确**：$a = -\omega^2 x$，加速度始终指向平衡位置

## 18 Electric Fields

### 1. 电场强度与电势混淆
- **错误**：认为 $E$ 和 $V$ 是同一种东西
- **正确**：$E$ 是矢量（电场强度），$V$ 是标量（电势）
- **关系**：$E = -\frac{dV}{dr}$（匀强电场中 $E = \frac{\Delta V}{\Delta d}$）

### 2. 库仑定律忘记分母的 $4\pi\epsilon_0$
- **错误**：$F = \frac{Q_1Q_2}{r^2}$（缺少 $1/4\pi\epsilon_0$）
- **正确**：$F = \frac{Q_1Q_2}{4\pi\epsilon_0 r^2}$
- **检查**：答案数量级是否在合理范围

## 19 Capacitance

### 1. 串并联电容公式和电阻搞混
- **错误**：电容串联用 $C = C_1 + C_2$
- **正确**：电容串联 $\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2}$，并联 $C = C_1 + C_2$（与电阻相反）

### 2. RC 放电指数公式写错
- **错误**：$x = x_0(1 - e^{-t/RC})$（这是充电）
- **正确**：放电 $x = x_0 e^{-t/RC}$，充电 $x = x_0(1 - e^{-t/RC})$

## 20 Magnetic Fields

### 1. 洛伦兹力方向判断错误
- **错误**：正负电荷受力方向搞反
- **正确**：弗莱明左手定则（正电荷方向），负电荷受力与正电荷相反
- **公式**：$F = BQv\sin\theta$，当 $v \perp B$ 时 $F = BQv$

### 2. 霍尔电压公式记错
- **错误**：$V_H = \frac{Bnq}{I}$ 或乱套公式
- **正确**：$V_H = \frac{BI}{ntq}$，其中 $n$ 是载流子数密度，$t$ 是导体厚度
- **提示**：$I = nAvq$，所以 $V_H = \frac{BI}{ntq} = \frac{Bv}{t/A}$... 记得 n 和 t 的物理意义

## 21 Alternating Currents

### 1. 有效值计算忘记除以 $\sqrt{2}$
- **错误**：$I_{rms} = \frac{I_0}{2}$
- **正确**：$I_{rms} = \frac{I_0}{\sqrt{2}}$
- **注意**：仅对正弦交流成立

## 22 Quantum Physics

### 1. 光电效应方程顺序搞错
- **错误**：$hf = \frac12 mv_{\max}^2 + \Phi$
- **正确**：$hf = \Phi + \frac12 mv_{\max}^2$（光子能量先用于克服逸出功，剩余为动能）
- **检查**：$hf < \Phi$ 时不会发生光电效应

### 2. 光子动量公式记错
- **错误**：$p = hf$
- **正确**：$p = \frac{h}{\lambda} = \frac{E}{c}$
- **注意**：动量只与波长有关，与频率不直接相关

## 23 Nuclear Physics

### 1. 质量亏损计算错误
- **错误**：直接用质量数减
- **正确**：质量亏损 = 核子质量和 $-$ 原子核实际质量
- **注意**：使用统一的原子质量单位 u（$1u = 1.66\times 10^{-27}\text{ kg}$）

### 2. 半衰期和衰变常数混淆
- **错误**：$\lambda = \frac{t_{1/2}}{0.693}$
- **正确**：$\lambda = \frac{0.693}{t_{1/2}}$ 或 $t_{1/2} = \frac{0.693}{\lambda}$

## 24 Medical Physics

### 1. 声阻抗公式忘记密度
- **错误**：$Z = c$
- **正确**：$Z = \rho c$（阻抗 = 密度 × 声速）

### 2. 强度反射系数符号
- **错误**：$\frac{I_r}{I_0} = \frac{(Z_1 - Z_2)^2}{(Z_1 - Z_2)^2}$（分子写成 $Z_1 + Z_2$）
- **正确**：$\frac{I_r}{I_0} = \frac{(Z_1 - Z_2)^2}{(Z_1 + Z_2)^2}$
- **意义**：$Z_1 \approx Z_2$ → 几乎无反射；$Z_1 \gg Z_2$ → 接近全反射

## 25 Astronomy and Cosmology

### 1. 斯特藩-玻尔兹曼定律幂次
- **错误**：$L \propto T^2$ 或 $L \propto T$
- **正确**：$L = 4\pi\sigma r^2 T^4$（与温度四次方成正比）

### 2. 红移公式正负号
- **错误**：红移时 $\Delta\lambda$ 为负
- **正确**：红移（远离）$\Delta\lambda > 0$（波长变长），蓝移（靠近）$\Delta\lambda < 0$

### 3. 光度距离公式
- **错误**：$F = \frac{L}{d^2}$（缺少 $4\pi$）
- **正确**：$F = \frac{L}{4\pi d^2}$

## 致命错误（不要犯！）

:::danger[最常见的错误]
1. **不写单位**：计算题答案必须有 SI 单位
2. **不写公式**：评分要求写出公式才给方法分
3. **"Show that" 题只写答案**：必须展示推导过程
4. **有效数字错误**：答案有效数字应和题目数据一致（至少 2–3 sig fig）
5. **温度用 $^\circ C$ 不转 $K$**：$pV = nRT$ 中的 $T$ 必须是开尔文
:::
