# 常见陷阱

## Trap 1: 忽略 AS 内容

Paper 4 明确考查 A Level 内容，但很多题目需要用到 AS 知识。

- **问题**：考到 SHM 中的弹簧 $F = kx$ 时忘记 $k$ 是弹簧常数（AS  Topic 6）
- **后果**：无法建立 $a = -\omega^2 x$ 与 $F = -kx$ 的联系
- **对策**：复习 AS Topic 1–11 的关键公式，特别是力学和电学基础

## Trap 2: 有效数字不匹配

- **问题**：题目给 $g = 9.81\text{ m s}^{-2}$（3 sig fig），但答案写 $9.8$ 或 $9.810$
- **后果**：答案被扣 SF 分
- **对策**：答案至少保留 3 sig fig；除非题目有 special instruction

## Trap 3: "Show that" 题只写结论

- **问题**：直接写"Therefore $v = \sqrt{2gh}$" 没有推导
- **后果**：0 分（需要展示从能量守恒到 $v = \sqrt{2gh}$ 的完整推导过程）
- **对策**：每一步写出所用公式和代入过程

## Trap 4: 万有引力公式乱用

- **问题**：地球表面的物体用 $F = GMm/r^2$ 但忘记 $r$ 是地心到物体的距离
- **后果**：用 $r = 0$ 或错误距离
- **注意**：$r$ 必须是 **centre-to-centre distance**，不是表面距离

## Trap 5: 霍尔电压公式中 $n$ 和 $t$ 的含义

- **问题**：$V_H = \frac{BI}{ntq}$ 中混淆 $n$（载流子数密度）和 $t$（导体厚度）
- **后果**：变量代入错误导致分数全扣
- **记忆**：$n$ 是 number density，$t$ 是 thickness（沿磁场方向的厚度）

## Trap 6: 热力学第一定律符号约定

- **问题**：$\Delta U = q + W$ 中 $W$ 的正负搞反
- **后果**：最终答案符号错误
- **对策**：系统对外做功 → $W$ 为负；外界对系统做功 → $W$ 为正

## Trap 7: 电容充放电混淆

- **问题**：放电题用 $x = x_0(1 - e^{-t/RC})$（充电公式）
- **后果**：完全算错
- **区分**：
  - 放电：$x = x_0e^{-t/RC}$（从初始值衰减到 0）
  - 充电：$x = x_0(1 - e^{-t/RC})$（从 0 升到终值）

## Trap 8: 交流有效值只对正弦波成立

- **问题**：对方波或其他波形也用 $I_{rms} = I_0/\sqrt{2}$
- **后果**：答案错误
- **注意**：$I_{rms} = I_0/\sqrt{2}$ 只对正弦（sinusoidal）交流成立

## Trap 9: 光电效应 $hf = \Phi + KE$ 的顺序

- **问题**：写成 $KE = hf + \Phi$
- **后果**：能量不守恒
- **记忆**：光子能量 `(hf)` = 克服逸出功 `(Φ)` + 剩余动能 `(½mv²)`

## Trap 10: 斯特藩-玻尔兹曼定律忘记 $4\pi$

- **问题**：$L = \sigma r^2 T^4$（缺少 $4\pi$）
- **后果**：数量级不对
- **正确**：$L = 4\pi\sigma r^2 T^4$（球表面积是 $4\pi r^2$）
