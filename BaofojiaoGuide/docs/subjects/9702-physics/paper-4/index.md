## 核心公式速查

| Topic | 核心公式 |
|-------|---------|
| Circular motion | $v = r\omega$, $a = r\omega^2 = \frac{v^2}{r}$, $F = m r \omega^2$ |
| Gravitation | $g = \frac{GM}{r^2}$, $\phi = -\frac{GM}{r}$, $E_p = -\frac{GMm}{r}$ |
| Ideal gases | $pV = nRT = NkT$, $\frac12 m\langle c^2 \rangle = \frac32 kT$ |
| Thermodynamics | $\Delta U = q + W$, $W = p\Delta V$ |
| SHM | $a = -\omega^2 x$, $x = x_0 \sin \omega t$, $v = \pm \omega\sqrt{x_0^2 - x^2}$ |
| Electric fields | $E = \frac{Q}{4\pi\epsilon_0 r^2}$, $V = \frac{Q}{4\pi\epsilon_0 r}$ |
| Capacitance | $C = \frac{Q}{V}$, $W = \frac12 CV^2$, $\tau = RC$, $x = x_0 e^{-t/RC}$ |
| Magnetic fields | $F = BIL\sin\theta$, $F = BQv\sin\theta$, $V_H = \frac{BI}{ntq}$ |
| EM induction | $\Phi = BA$, $E = -N\frac{d\Phi}{dt}$ |
| AC | $x = x_0 \sin \omega t$, $I_{rms} = \frac{I_0}{\sqrt2}$, $V_{rms} = \frac{V_0}{\sqrt2}$ |
| Quantum | $E = hf = \frac{hc}{\lambda}$, $p = \frac{h}{\lambda}$, $hf = \Phi + \frac12 mv^2_{\max}$ |
| Nuclear | $\Delta E = \Delta m c^2$, $A = \lambda N$, $t_{\frac12} = \frac{0.693}{\lambda}$, $x = x_0 e^{-\lambda t}$ |
| Medical | $Z = \rho c$, $\frac{I_r}{I_0} = \frac{(Z_1 - Z_2)^2}{(Z_1 + Z_2)^2}$, $I = I_0 e^{-\mu x}$ |
| Astronomy | $F = \frac{L}{4\pi d^2}$, $\lambda_{\max} T = \text{constant}$, $L = 4\pi\sigma r^2 T^4$, $v \approx H_0 d$ |

## 整体策略

- **计算题占比最大（~40%）**，必须熟练掌握所有公式及其变式
- **解释题（~30%）** 需要用精确的关键词作答，可参考 Mark Scheme 的采分点
- **定义题（~15–20%）** 用一句话准确说出定义，不要遗漏关键术语
- **作图题（~8–10%）** 注意坐标轴标注、关键点、渐近线
- **"Show that" 题** 必须写出完整的推导步骤，不能只写答案
