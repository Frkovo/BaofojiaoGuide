# 题型总览

六大核心题型，覆盖全部 5 个知识板块。

## Type 1: CRV 概率计算（PDF/CDF 互求）

| 项目 | 内容 |
|------|------|
| 描述 | 给出 PDF 求 CDF、概率、中位数、众数、期望、方差，或反求归一化常数 |
| 常见子问 | (a) 求 $k$ 使 $\int f = 1$，5.1 (b) 求 $F(x)$，5.2 (c) 求 $E(X)$，5.4 (d) 求中位数，5.4 |
| 典型分值 | 8–10 分 |
| 关联板块 | 4.1 CRV |
| 关键技巧 | 分段积分、求导 CDF、解 $F(m)=0.5$ |
| 常见难度 | 中等，但积分错误率较高 |

## Type 2: 参数假设检验（z 检验 / t 检验）

| 项目 | 内容 |
|------|------|
| 描述 | 使用样本数据对总体参数进行假设检验或构造置信区间 |
| 子类型 | 单样本 t / 双样本 t（等方差）/ 配对 t / z 检验 |
| 典型分值 | 8–10 分 |
| 关联板块 | 4.2 推断 |
| 关键技巧 | 正确选择检验类型、合并方差、$H_0$/$H_1$、自由度 |
| 常见难度 | 中等，关键在于选择正确的检验 |

## Type 3: 卡方检验（拟合优度 / 独立性）

| 项目 | 内容 |
|------|------|
| 描述 | 拟合优度检验（观测是否符合某理论分布）或列联表独立性检验 |
| 子类型 | 拟合优度（含参数估计）、列联表、$2\times2$（Yates 校正） |
| 典型分值 | 8–10 分 |
| 关联板块 | 4.3 卡方 |
| 关键技巧 | 期望频数计算、自由度确定、合并条件、Yates 校正 |
| 常见难度 | 中等偏易，但自由度陷阱多 |

## Type 4: 非参数检验（Wilcoxon / Mann-Whitney / Sign）

| 项目 | 内容 |
|------|------|
| 描述 | 使用秩或符号进行的无分布假设检验 |
| 子类型 | Wilcoxon 符号秩（单样本/配对）、Mann-Whitney U（双样本）、符号检验 |
| 典型分值 | 6–9 分 |
| 关联板块 | 4.4 非参数 |
| 关键技巧 | 排序、求秩、打结处理、正态近似（含连续性校正） |
| 常见难度 | 中等，排序过程繁琐但得分率高 |

## Type 5: PGF 计算与分布识别

| 项目 | 内容 |
|------|------|
| 描述 | 求 PGF、利用 PGF 求期望方差、独立和/随机和的 PGF |
| 子类型 | 求标准分布 PGF、PGF 求矩、独立和、随机和 |
| 典型分值 | 7–9 分 |
| 关联板块 | 4.5 PGF |
| 关键技巧 | 正确求导、代入 $t=1$、区分独立和与随机和 |
| 常见难度 | 中等偏上，容易混淆公式 |

## Type 6: 综合应用题（跨板块）

| 项目 | 内容 |
|------|------|
| 描述 | 结合两个或以上知识板块的综合大题，通常在试卷最后 |
| 常见组合 | CRV + PGF、CRV + 推断、PGF + 推断、非参数 + 推断 |
| 典型分值 | 10–12 分 |
| 关联板块 | 多个板块 |
| 关键技巧 | 分步处理、明确每一步在考哪个板块、写出过渡文字 |
| 常见难度 | 较难，需灵活运用多个知识体系 |

## 题型与分值汇总

| 题型 | 典型分值 | 出现频次 | 在试卷中的位置 |
|------|---------|---------|--------------|
| Type 1: CRV 计算 | 8–10 | 1–2 题 | Q2–Q3 或 Q6 |
| Type 2: 参数检验 | 8–10 | 1–2 题 | Q1 或 Q6 |
| Type 3: 卡方检验 | 8–10 | 1 题 | Q2–Q3 |
| Type 4: 非参数检验 | 6–9 | 1 题 | Q1 或 Q4 |
| Type 5: PGF | 7–9 | 1 题 | Q4–Q5 |
| Type 6: 综合题 | 10–12 | 0–1 题 | Q7（若有） |

## 备考顺序建议

1. **Type 3（卡方）** — 套路最固定，最容易拿满
2. **Type 4（非参数）** — 步骤规范，算对秩即可得分
3. **Type 2（参数检验）** — 关键在选择正确的检验
4. **Type 1（CRV）** — 积分能力决定成败
5. **Type 5（PGF）** — 公式易混淆，需重点练习
6. **Type 6（综合）** — 基础扎实后自然能应对

:::info[评分标准（MS 模式）]
每道综合题的评分通常是**分步评分**。即使最终答案不正确，只要中间步骤正确（如正确写出 $H_0$、正确代入公式），仍然可以获得 **M1** / **B1** 方法分。**不要因为做不出最终答案而留白**，写出你能写的每一步。
:::
