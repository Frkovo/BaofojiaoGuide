# Mark Scheme Patterns — Non-parametric Tests

## Wilcoxon Signed-Rank 给分模式

| 步骤 | 标记 | 说明 |
|------|------|------|
| 正确提出 $H_0$, $H_1$ | **B1** | 含中位数和检验方向 |
| 计算差值 | **M1** |  |
| 给绝对值排序并赋秩 | **M1** | 方法分 |
| 秩全部正确 | **A1** | 所有秩值正确 |
| 计算 $T^+$ 和 $T^-$ | **M1** |  |
| $T$ 值正确 | **A1** | $\min(T^+, T^-)$ |
| 查表得临界值 | **B1** | 使用正确 $n$ 和 $\alpha$ |
| 比较并结论 | **M1 A1** | 比较 + 上下文 |

## Wilcoxon Rank-Sum 给分模式

| 步骤 | 标记 | 说明 |
|------|------|------|
| 正确提出 $H_0$, $H_1$ | **B1** |  |
| 合并并排序所有数据 | **M1** |  |
| 秩全部正确 | **A1** | 注意 ties 取平均 |
| 计算 $R_1$ | **M1** |  |
| $W$ 值正确 | **A1** |  |
| 查表得临界值 | **B1** | 双边时注意上下界 |
| 比较并结论 | **M1 A1** |  |

## Sign Test 给分模式

| 步骤 | 标记 | 说明 |
|------|------|------|
| 提出假设 | **B1** |  |
| 计算符号数 | **M1** | 排除 0 |
| $S$ 和 $n$ 正确 | **A1** |  |
| 计算二项概率 | **M1** | $P(S \leq s)$ |
| p-value 正确 | **A1** | 双尾时 $\times 2$ |
| 比较并结论 | **M1 A1** |  |

## 正态近似给分模式

| 步骤 | 标记 | 说明 |
|------|------|------|
| 计算 $\mu$ | **B1** | 公式正确 |
| 计算 $\sigma$ | **M1** | 公式正确 |
| 计算 $z$ | **M1** | 含连续性校正 |
| $z$ 值正确 | **A1** |  |
| 比较 $|z|$ 与临界值 | **M1** |  |
| 结论 | **A1** |  |

## 常见给分模式总结

- Wilcoxon signed-rank 共约 8-10 分
- Wilcoxon rank-sum 共约 8-10 分  
- Sign test 共约 6-8 分
- 正态近似通常作为某大题的一部分（4-6 分）
- 临界值查表通常给 **B1**，但若题目直接在表中给出则可能不给
- 连续性校正通常为 **M1**（方法分），不做或做错扣 1 分
