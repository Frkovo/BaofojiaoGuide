# 常见错误 — Non-parametric Tests

## 错误 1：混淆 $T^+$、$T^-$ 和 $T$

**错误**：直接用 $T^+$ 或 $T^-$ 与临界值比较

**正解**：检验统计量 $T = \min(T^+, T^-)$，用较小的秩和与临界值比较

**例外**：单侧检验时，如果备择假设指定方向（如 median &gt; $m_0$），则用 $T^-$（负秩和）作为检验统计量，但 9231 大多用双尾，建议统一用 $T = \min(T^+, T^-)$

## 错误 2：忽略零差异

**错误**：对 $d_i = 0$ 也赋予秩

**正解**：零差异排除在外，$n$ 只算非零差异的个数

## 错误 3：秩排错方向

**错误**：最大差异给秩 1

**正解**：最小绝对差异给秩 1（从小到大排序）

## 错误 4：Ties（相同值）的处理

**错误**：相同值赋予相同秩

**正解**：相同值取平均秩。例如两个相同的第 3、4 位值，都取秩 $(3+4)/2 = 3.5$

## 错误 5：忘记符号检验的对称性假设

**错误**：数据明显不对称时仍用 Wilcoxon signed-rank

**正解**：Wilcoxon signed-rank 要求差异分布**对称**。若不对称，改用 Sign test

## 错误 6：正态近似忘记连续性校正

**错误**：$z = \dfrac{T - \mu}{\sigma}$ 不含 $+0.5$

**正解**：必须包含连续性校正：$z = \dfrac{T \pm 0.5 - \mu}{\sigma}$
- 下尾：$z = \dfrac{T + 0.5 - \mu}{\sigma}$
- 上尾：$z = \dfrac{T - 0.5 - \mu}{\sigma}$

## 错误 7：Sign test 忘记双尾 p-value

**错误**：只算 $P(S \leq s)$ 就与 $\alpha$ 比较

**正解**：双尾检验时 p-value $= 2 \times P(S \leq s)$（当 $s &lt; n/2$）

## 错误 8：Rank-sum test 样本大小标注错误

**错误**：混淆 $n_1$ 和 $n_2$ 导致查表错误

**正解**：$n_1$ 和 $n_2$ 的标注必须与查表一致。题目中列出的第一个样本通常是 $n_1$

## 错误 9：查表用错 $\alpha$

**错误**：双尾检验用单尾临界值

**正解**：双尾检验 $\alpha = 0.05$ 时，查表用 $\alpha = 0.025$（每尾）。注意表头标注的是单尾还是双尾

## 错误 10：混淆 Wilcoxon 和 Sign test 的使用场景

| 场景 | 正确选择 |
|------|---------|
| 对称分布 + 配对数据 | Wilcoxon signed-rank（更灵敏） |
| 不对称分布 + 配对数据 | Sign test |
| 仅需知道方向 | Sign test（更简单） |

## 错误 11：Sign test 计算二项概率时方向搞反

**错误**：不清楚 $P(S \geq s)$ 还是 $P(S \leq s)$

**正解**：若 $s &lt; n/2$，计算 $P(S \leq s)$；若 $s &gt; n/2$，计算 $P(S \geq s) = 1 - P(S \leq s-1)$。p-value 是较小一侧概率的两倍（双尾）
