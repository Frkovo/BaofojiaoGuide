# 常见错误 — Probability Generating Functions

## 错误 1：混淆 $G'(1)$ 与 $E[X^2]$

**错误**：认为 $G'(1) = E[X^2]$。

**正解**：$G'(1) = E[X]$，$E[X^2] = G''(1) + G'(1)$。

$G(t) = E[t^X]$，其一阶导在 $t=1$ 处给出 $E[X]$，而非 $E[X^2]$。

## 错误 2：方差公式记错

**错误**：$\text{Var}(X) = G''(1) - [G'(1)]^2$（遗漏 $+G'(1)$）。

**正解**：$\text{Var}(X) = \boxed{G''(1) + G'(1) - [G'(1)]^2}$。

推导：$\text{Var}(X) = E[X^2] - (E[X])^2 = [E[X(X-1)] + E[X]] - (E[X])^2$
$= G''(1) + G'(1) - [G'(1)]^2$

## 错误 3：展开后不合并同类项

**错误**：写出 $G(t)$ 后不化简，导致后续求导或读系数时出错。

**正解**：构造 PGF 后合并 $t^k$ 的同类项，确保每个 $t^k$ 只有一个系数。

如 $G(t) = \frac{2}{5} + \frac{8}{15}t + \frac{1}{15}t^2$（已化简），而非 $G(t) = \frac{C^4_2}{C^6_2} + \frac{C^4_1C^2_1}{C^6_2}t + \frac{C^2_2}{C^6_2}t^2$。

## 错误 4：不放回抽样概率计算错误

**错误**：用二项分布概率（有放回）计算不放回问题。

**正解**：不放回抽样使用超几何概率——组合数计算：

$$
P(X = x) = \frac{C^{\text{成功}}_x \times C^{\text{失败}}_{n-x}}{C^{\text{总数}}_n}
$$

如袋中 4 红 2 蓝，取 2 球不放回，蓝球数 $X$：
- $P(X=0) = \frac{C^4_2}{C^6_2} = \frac{6}{15}$，而非 $\binom{2}{0}(\frac{2}{6})^0(\frac{4}{6})^2$

## 错误 5：和的 PGF 写成和

**错误**：$G_{X+Y}(t) = G_X(t) + G_Y(t)$。

**正解**：独立随机变量和的 PGF 是各 PGF 的**乘积**：

$$
G_{X+Y}(t) = G_X(t) \cdot G_Y(t)
$$

当 $n$ 个独立同分布变量求和时：

$$
G_{S_n}(t) = [G_X(t)]^n \quad (\text{不是 } n \cdot G_X(t))
$$

## 错误 6：几何分布的定义混淆

**错误**：使用错误的几何分布 PGF 公式。

**正解**：9231 考纲中若 $X$ 表示首次成功所需的试验次数（$x = 1, 2, \dots$）：

$$
G_X(t) = \frac{pt}{1 - qt},\quad q = 1-p
$$

若 $X$ 表示首次成功前的失败次数（$x = 0, 1, \dots$），则 $G_X(t) = \frac{p}{1 - qt}$。注意题目中的定义。

## 错误 7：反推分布时开方符号出错

**错误**：$G_S(t) = [G_X(t)]^2$ 开平方时忘记取正根。

**正解**：概率生成函数系数非负，只取正根。

如 $[G_X(t)]^2 = \frac{1}{4}(1+t)^2 \Rightarrow G_X(t) = \frac{1}{2}(1+t)$（而非 $-\frac{1}{2}(1+t)$）。

## 错误 8：不验证系数合理性

**错误**：得到 $G_X(t)$ 后不检查系数是否在 $[0,1]$ 范围内、是否和为 1。

**正解**：逆向题中求出 $G_X(t)$ 后应验证 $\sum P(X=x) = G_X(1) = 1$。

## 错误 9：求导时链式法则遗漏

**错误**：对 $(a+bt)^n$ 求导时忘记乘以内层导数 $b$。

**正解**：$\frac{d}{dt}(a+bt)^n = n(a+bt)^{n-1} \cdot b$。例如 $\frac{d}{dt}(\frac{1}{2}+\frac{1}{2}t)^4 = 4(\frac{1}{2}+\frac{1}{2}t)^3 \cdot \frac{1}{2}$。

## 错误 10：混淆 $G(t)$ 的参数形式

**错误**：将 $G_X(t) = (q+pt)^n$ 用于 $n=0$ 的情况或误解 $p$。

**正解**：确认参数含义。$q = 1-p$，$n$ 为试验次数，$p$ 为成功概率。代入前检查参数是否合理。
