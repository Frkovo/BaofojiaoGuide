# 考前速记 — Probability Generating Functions

## 核心公式

| 公式 | 说明 |
|------|------|
| $G_X(t) = E[t^X] = \sum P(X=x)t^x$ | PGF 定义 |
| $G_X(1) = 1$ | 总概率为 1 |
| $G_X(0) = P(X = 0)$ | 常数项 |
| $E[X] = G'(1)$ | 均值 |
| $E[X(X-1)] = G''(1)$ | 阶乘矩 |
| $\text{Var}(X) = G''(1) + G'(1) - [G'(1)]^2$ | 方差 |
| $G_{X+Y}(t) = G_X(t) G_Y(t)$ | 独立变量和的 PGF |
| $G_{S_n}(t) = [G_X(t)]^n$ | $n$ 个 i.i.d. 变量和 |

## 标准分布 PGF 速查

| 分布 | $G(t)$ | $E[X]$ | $\text{Var}(X)$ |
|------|--------|--------|-----------------|
| $\text{DU}(n)$ | $\frac{1}{n}(t + t^2 + \cdots + t^n)$ | $\frac{n+1}{2}$ | $\frac{n^2-1}{12}$ |
| $B(n, p)$ | $(q+pt)^n$ | $np$ | $npq$ |
| $\text{Geo}(p)$ | $\frac{pt}{1-qt}$ | $\frac{1}{p}$ | $\frac{q}{p^2}$ |
| $\text{Po}(\lambda)$ | $e^{\lambda(t-1)}$ | $\lambda$ | $\lambda$ |

## 解题流程

### 构造 PGF

读题 → 确定 $X$ 取值 → 计算各 $P(X=x)$ → 写 $G(t) = \sum P(X=x)t^x$

### 求均值和方差

$G(t)$ → $G'(t)$ → $G'(1) = E[X]$ → $G''(t)$ → $G''(1) = E[X(X-1)]$ → $\text{Var} = G''(1) + G'(1) - [G'(1)]^2$

### 求和的概率

各 $G$ 相乘 → 展开 → $t^k$ 系数 = $P(\text{sum} = k)$

### 反推分布

$G_S(t) = [G_X(t)]^n$ → 开 $n$ 次方得 $G_X(t)$ → 读系数

## 易错点提醒

- $G'(1) = E[X]$，**不是** $E[X^2]$
- 方差公式有 $+G'(1)$，不是只有 $G''(1) - [G'(1)]^2$
- 和的 PGF 是**乘积**，不是和
- 不放回用组合数，不放回用组合数，不放回用组合数（重要的事说三遍）
- 几何分布注意定义（从 $x=1$ 还是 $x=0$ 开始）
- 开方求 $G_X(t)$ 只取正根
- 化简多项式后再求导
