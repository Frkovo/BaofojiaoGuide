# 考纲要求 — Chi-squared Tests

## 1. $\chi^2$ 检验基本原理

- 观测频数 $O$ 与期望频数 $E$ 的比较
- 统计量 $\chi^2 = \sum \dfrac{(O - E)^2}{E}$ 服从（近似）$\chi^2$ 分布
- 大样本条件：每个 $E \geq 5$

## 2. 拟合优度检验（GOF）

- 检验数据是否符合某个特定分布（Binomial、Poisson 或给定 PDF）
- 步骤：
  1. 提出 $H_0$（数据服从指定分布）和 $H_1$（数据不服从指定分布）
  2. 计算每个类别的期望频数 $E = n \times P(X = x)$
  3. 若需要从数据估计分布参数（如 Binomial 的 $p$ 或 Poisson 的 $\lambda$），则先估计
  4. 合并 $E &lt; 5$ 的类别
  5. 计算 $\chi^2$ 统计量
  6. 自由度 $\mathrm{df} = k - 1 - m$，其中 $m$ 是估计的参数个数
  7. 与临界值比较，得结论

## 3. 独立性检验（Contingency Tables）

- 检验两个分类变量是否独立
- 步骤：
  1. 提出 $H_0$（变量独立）和 $H_1$（变量不独立）
  2. 计算每格期望频数 $E_{ij} = \dfrac{R_i \times C_j}{T}$
  3. 合并 $E &lt; 5$ 的单元格（整行或整列合并）
  4. 计算 $\chi^2$ 统计量
  5. 自由度 $\mathrm{df} = (r - 1)(c - 1)$
  6. 与临界值比较，得结论

## 4. 期望频数合并规则

- 任何预期频数 $E &lt; 5$ 的单元格必须与相邻单元格合并
- 合并后重新计算 $\chi^2$
- 合并后自由度相应减少（类别数 $k$ 减少）

## 5. 不考内容

- Yates 连续性校正
- $\chi^2$ 检验用于比例比较（已从考纲移除）
