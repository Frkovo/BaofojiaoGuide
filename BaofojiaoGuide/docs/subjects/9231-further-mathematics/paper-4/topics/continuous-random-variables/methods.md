# 解题方法 — Continuous Random Variables

## Method 1: Finding Constant from PDF

**适用场景**：PDF 含未知常数 $k$，需要利用总概率为 1 求解。

:::note[标准解题步骤]

1. 写出 PDF 的非零区间
2. 对 PDF 在整个定义域积分：$\displaystyle \int_{-\infty}^{\infty} f(x)\,dx = 1$
3. 对于分段 PDF，在每一段上分别积分再求和
4. 解方程得 $k$
5. 验证 $f(x) \ge 0$（若 $k$ 使某段为负则舍弃）

:::

:::info[评分标准（MS 模式）]

- **B1** 写出 $\int f(x)\,dx = 1$
- **M1** 正确积分（含上下限）
- **A1** $k$ 值正确

:::

:::warning[常见陷阱]

- 忘记积分必须覆盖所有非零区域
- 分段积分时漏掉某一段
- 求出的 $k$ 导致 $f(x) &lt; 0$ 未检查

:::

## Method 2: Finding CDF from PDF

**适用场景**：已知 PDF $f(x)$，求 $F(x) = P(X \le x)$。

:::note[标准解题步骤]

1. 确定 PDF 的非零区域
2. 对 $x$ 分类讨论（分段 PDF 需分段积分）
3. 对每段，从下限积到 $x$：$F(x) = \int_{-\infty}^{x} f(t)\,dt$
4. 验证 $F(-\infty) = 0$ 和 $F(\infty) = 1$
5. 写出分段 CDF 表达式

:::

:::info[评分标准（MS 模式）]

- **M1** 对所有 $x$ 写出正确的积分形式
- **M1** 对每一段分别积分
- **A1** 每一段的 $F(x)$ 表达式正确
- **B1** 验证端点条件

:::

:::warning[常见陷阱]

- 积分常数错误（未用 $F(-\infty) = 0$ 确定常数）
- 分段的边界点处 $F$ 不连续
- 某一段的积分起点错误

:::

## Method 3: Finding $E(X)$ and $\text{Var}(X)$

**适用场景**：已知 PDF 或 CDF，求均值和方差。

:::note[标准解题步骤]

1. $E[X] = \int_{-\infty}^{\infty} x f(x)\,dx$
2. $E[X^2] = \int_{-\infty}^{\infty} x^2 f(x)\,dx$
3. $\text{Var}(X) = E[X^2] - (E[X])^2$
4. 也可用 $\text{Var}(X) = E[(X - \mu)^2]$ 直接积分

:::

:::info[评分标准（MS 模式）]

- **M1** 写出 $E[X] = \int x f(x)\,dx$
- **A1** $E[X]$ 正确
- **M1** 写出 $E[X^2] = \int x^2 f(x)\,dx$
- **A1** $E[X^2]$ 正确
- **M1** $\text{Var}(X) = E[X^2] - (E[X])^2$
- **A1** $\text{Var}(X)$ 正确

:::

:::warning[常见陷阱]

- 分段 PDF 未在每段正确乘以 $x$ 或 $x^2$
- 方差公式写成 $E[X^2] - E[X]$
- 忘记给出最终数值（或分数形式）

:::

## Method 4: Finding Median and Quartiles

**适用场景**：已知 CDF 或 PDF，求中位数（median）、上下四分位数（$Q_1, Q_3$）、百分位数。

:::note[标准解题步骤]

1. 先由 PDF 求出 CDF $F(x)$（若未给出）
2. 对中位数解 $F(m) = 0.5$
3. 对 $Q_1$ 解 $F(Q_1) = 0.25$，对 $Q_3$ 解 $F(Q_3) = 0.75$
4. 对 $\alpha$ 百分位解 $F(p_\alpha) = \alpha/100$
5. 检查解是否在对应分段区间内

:::

:::info[评分标准（MS 模式）]

- **M1** 写出 $F(m) = 0.5$ 或 $F(Q_1) = 0.25$
- **M1** 正确代入 CDF 表达式
- **A1** 数值正确（精确值或 3 s.f.）

:::

:::warning[常见陷阱]

- 未确认解落在正确的分段区间中
- 在不含有中位数的分段上求解
- 忘记对分位数值开方（当 CDF 含平方项时）

:::

## Method 5: Transformation $Y = g(X)$ (CDF Method)

**适用场景**：已知 $X$ 的分布，求 $Y = g(X)$ 的 PDF。

:::note[标准解题步骤]

1. 写出 CDF：$F_Y(y) = P(Y \le y) = P(g(X) \le y)$
2. 解不等式 $g(X) \le y$ 得到 $X$ 的范围
3. 用 $X$ 的 CDF 表示：$F_Y(y) = F_X(\text{upper}) - F_X(\text{lower})$（或积分）
4. 对 $y$ 求导得 PDF：$f_Y(y) = F_Y'(y)$
5. 注明 $y$ 的取值范围

:::

:::info[评分标准（MS 模式）]

- **M1** 写出 $F_Y(y) = P(g(X) \le y)$
- **M1** 正确转换不等式得 $X$ 的范围
- **M1** 写出用 $F_X$ 或积分表示的概率
- **A1** $F_Y(y)$ 形式正确
- **M1** 对 $y$ 求导
- **A1** $f_Y(y)$ 正确，含定义域

:::

:::warning[常见陷阱]

- 忽略 $g$ 的单调性（单调 / 非单调需分开处理）
- 忘记 $Y$ 的取值范围（如 $Y = X^2$ 得 $y \ge 0$）
- 非单调变换（如 $Y = X^2$ 在对称区间）未正确处理正负区间
- 对 $y$ 求导时链式法则错误

:::

## Method 6: Piecewise PDF Problems

**适用场景**：PDF 在不同区间有不同的表达式，需要综合运用各种方法。

:::note[标准解题步骤]

1. 识别各分段区间
2. 用总积分 = 1 求常数（如果未知）
3. 逐段求 CDF，注意连续性
4. 计算期望时分段积分 $E[X] = \int_{x_1}^{x_2} x f_1(x)\,dx + \int_{x_2}^{x_3} x f_2(x)\,dx + \dots$
5. 求中位数时先确定中位数所在的分段区间
6. 画出示意图帮助理解

:::

:::info[评分标准（MS 模式）]

- **B1** 识别分段并正确写出各段积分
- **M1** 对每段正确积分
- **A1** 每段结果正确
- **M1** 将各段结果求和
- **A1/F1** 最终答案正确（follow-through 可能适用）

:::

:::warning[常见陷阱]

- 分段积分时用错极限（每段的上限或下限错误）
- $E[X^2]$ 的积分中每段各自乘 $x^2$
- 中位数所在区间判断错误
- 变换 $Y=g(X)$ 时未考虑分段边界

:::
