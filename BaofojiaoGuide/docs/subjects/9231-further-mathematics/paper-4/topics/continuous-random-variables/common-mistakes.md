# 常见错误 — Continuous Random Variables

## 错误 1：求常数时积分区间错误

**描述**：对分段 PDF 积分时，未对每段分别积分再求和，或积分上下限用错。

**错误示例**：$f(x) = \begin{cases} kx, & 0 \le x \le 1 \\ k(2-x), & 1 &lt; x \le 2 \end{cases}$ 积分时写成 $\int_0^2 kx\,dx$

**正确做法**：$\int_0^1 kx\,dx + \int_1^2 k(2-x)\,dx = 1$

## 错误 2：CDF 积分常数错误

**描述**：求 CDF 时不定积分忘记加常数，或未用 $F(-\infty) = 0$ 确定常数。

**正确做法**：对每段从下限积到 $x$，即 $F(x) = \int_{-\infty}^x f(t)\,dt$，不是 $\int f(x)\,dx$

## 错误 3：方差公式记错

**描述**：$\text{Var}(X) = E[X^2] - E[X]$（漏了平方）

**正确公式**：$\text{Var}(X) = E[X^2] - (E[X])^2$

## 错误 4：中位数/四分位数未检查区间

**描述**：解 $F(m) = 0.5$ 后，未验证 $m$ 是否在所用的分段区间内。

**正确做法**：先确定中位数所在区间（如 $F(a) &lt; 0.5 &lt; F(b)$），再用该段的 $F$ 表达式求解。

## 错误 5：变换 $Y = X^2$ 时忽略正负

**描述**：$Y = X^2$ 时，$P(Y \le y) = P(-\sqrt{y} \le X \le \sqrt{y})$，但只写 $P(X \le \sqrt{y})$。

**正确做法**：$F_Y(y) = P(X^2 \le y) = P(-\sqrt{y} \le X \le \sqrt{y}) = F_X(\sqrt{y}) - F_X(-\sqrt{y})$

## 错误 6：变换后忘记定义域

**描述**：求 $Y = g(X)$ 的 PDF 后，未注明 $y$ 的取值范围。

**正确做法**：$Y$ 的取值范围由 $g$ 和 $X$ 的定义域共同决定，必须明确写出。

## 错误 7：对 $P(X = a)$ 的理解

**描述**：在连续随机变量中认为 $P(X = a) \ne 0$。

**正确理解**：连续随机变量中 $P(X = a) = 0$，因此 $P(a \le X \le b) = P(a &lt; X &lt; b)$。

## 错误 8：分段积分求 $E[X^2]$ 时忘记乘 $x^2$

**描述**：各段积分仍使用 $x f(x)$ 而非 $x^2 f(x)$。

**正确做法**：$E[X^2] = \int x^2 f(x)\,dx$，对每段分别乘 $x^2$ 再积分。

## 错误 9：CDF 求导为 PDF 时忽略链式法则

**描述**：对 $F_Y(y) = F_X(\sqrt{y})$ 求导时，忘记乘以 $\frac{d}{dy}\sqrt{y} = \frac{1}{2\sqrt{y}}$。

**正确做法**：$f_Y(y) = f_X(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}}$

## 错误 10：未验证 PDF 非负性

**描述**：求出常数 $k$ 后未检查 $f(x) \ge 0$ 对所有 $x$ 成立。

**正确做法**：若 $k$ 使某段 PDF 为负，则该值不合理，需重新检查计算。

## 错误 11：$E[g(X)]$ 计算中混淆 $g$ 的复合形式

**描述**：求 $E[\sin X]$ 时写成 $\sin \int x f(x)\,dx$，而非 $\int \sin x \cdot f(x)\,dx$。

**正确做法**：$E[g(X)] = \int g(x) f(x)\,dx$，$g$ 作用在 $x$ 上，不是作用在期望上。

## 错误 12：求 CDF 时分类讨论遗漏

**描述**：分段 PDF 的 $F(x)$ 需要分 $x$ 在左端左侧、各段内部、右端右侧共 $n+1$ 种情况，容易遗漏。

**正确做法**：对 $n$ 段 PDF，$F(x)$ 至少有 $n+1$ 种情况（左端外、每段内、右端外）。
