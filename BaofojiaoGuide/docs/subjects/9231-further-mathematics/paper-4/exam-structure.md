| 时长 | 1 小时 30 分钟（90 分钟） |
| 总分 | **50** 分 |

| 题号 | 常见板块 | 常见考查方式 |
|------|---------|-------------|
| Q1 | 4.2 推断 / 4.4 非参数 | t 检验（单样本/双样本/配对），或 Wilcoxon / Mann-Whitney 检验 |
| Q2 | 4.3 卡方 / 4.1 CRV | 拟合优度或列联表检验；或 CRV 求 CDF 与概率 |
| Q3 | 4.1 CRV / 4.3 卡方 | CRV 中位数/众数/期望；或卡方检验 |
| Q4 | 4.4 非参数 / 4.5 PGF | 符号秩检验 / Mann-Whitney；或 PGF 求期望方差 |
| Q5 | 4.5 PGF / 4.4 非参数 | PGF 独立和与随机和；或 Wilcoxon 检验 |
| Q6 | 4.1 CRV / 4.2 推断 | 较复杂的 CRV（含绝对值/线性变换）；或综合推断题 |
| Q7 | 综合题 | 跨板块综合题（如 CRV + PGF，或 CRV + 推断） |

:::note[考试特点]

:::

## 作答规范

### 假设检验的标准答题步骤

1. **明确 $H_0$ 和 $H_1$**（需用文字或符号清晰说明）
2. **选择显著性水平 $\alpha$**（题目给或不给，通常 5%）
3. **计算检验统计量**（需写出公式并代入数值）
4. **确定临界值或 $p$ 值**（注明自由度 df）
5. **比较并作结论**（`Reject $H_0$` 或 `Do not reject $H_0$`）
6. **用文字解释结论**（在题目情境下说明含义）

:::danger[致命错误]
**缺少 $H_0$/$H_1$ 定义**，或 **结论未在语境中解释**，将直接丢失 **M1** 甚至整题分数。
:::

### 答案格式

- 所有概率表示为小数（通常保留 3 位有效数字）或精确分数
- 所有 $p$ 值保留至可比较的程度（通常 3–4 位小数）
- 检验统计量保留 2–3 位小数
- 置信区间写作 $(a, b)$ 形式
- 明确写出单位（如适用）

### Marking Scheme 符号解读

| 符号 | 含义 | 说明 |
|------|------|------|
| **M1** | Method mark（方法分） | 使用了正确的方法，即使算术错误也可得分 |
| **A1** | Accuracy mark（精度分） | 在前述方法正确的前提下结果正确 |
| **B1** | Independent mark（独立分） | 不需要依赖前一步，独立给分 |
| **M1 A1 FT** | Follow through（跟随分） | 即使前一步错了，后续步骤正确也能得分 |
| **AWFW** | Anything which fits within（合理范围内） | 允许计算器舍入差异的容差范围 |
| **CAO** | Correct answer only（仅正确答案） | 仅当最终答案正确时才给分 |
| **OE** | Or equivalent（或等价形式） | 等价的表达式也可接受 |
| **ISW** | Ignore subsequent working（忽略后续工作） | 后续错误不影响已经得到的分数 |
