# MS关键词速查

## Common Mark Types（通用标记类型）

| MS 标记 | 全称 | 中文含义 | 说明 |
|---------|------|---------|------|
| **M1** | Method mark | 方法分 | 使用了正确方法即可得分，不要求最终答案正确 |
| **A1** | Accuracy mark | 精度分 | 在前一步方法正确的前提下，答案正确 |
| **B1** | Independent mark | 独立给分 | 不依赖前序步骤，独立得分（如正确写出 $H_0$ / $H_1$） |
| **M1 A1 FT** | Follow through | 跟随给分 | 即使前序步骤有误，后续计算若基于该错误值正确也可得分 |
| **DM1** | Dependent method | 依赖方法分 | 仅当满足前置条件时才给方法分 |
| **DB1** | Dependent B mark | 依赖 B 分 | 仅当满足前置条件时才给的独立分 |

## Modifiers（限定修饰词）

| MS 短语 | 中文含义 | 示例 |
|---------|---------|------|
| CAO | Correct Answer Only | 仅最终答案正确才给 **A1** |
| AWFW | Anything Which Fits Within | 答案在合理范围内即可（通常 ±0.005） |
| ISW | Ignore Subsequent Working | 后续错误工作忽略，不影响已得分 |
| OE | Or Equivalent | 等价形式也可接受 |
| AG | Answer Given | 答案已在题目中给出，必须推导证明 |
| SC | Special Case | 特殊情况下的特殊评分规则 |
| AWRT | Anything Which Rounds To | 四舍五入到指定位数后正确即可 |

## 结论措辞模板

| 情形 | MS 接受的结论措辞 | 中文对应 |
|------|------------------|---------|
| 拒绝 $H_0$ | "Reject $H_0$" / "There is sufficient evidence" | 拒绝 $H_0$，有充分证据 |
| 不拒绝 $H_0$ | "Do not reject $H_0$" / "Insufficient evidence" | 不拒绝 $H_0$，证据不足 |
| 显著 | "Significant at $\alpha\%$ level" | 在 $\alpha\%$ 水平上显著 |
| 不显著 | "Not significant at $\alpha\%$ level" | 在 $\alpha\%$ 水平上不显著 |
| $p$ 值法 | "$p\text{-value} < \alpha$, reject $H_0$" | $p$ 值小于 $\alpha$，拒绝 $H_0$ |

:::note[MS 实战提示]
- MS 中出现的 **"OE"**（or equivalent）意味着你的表达式不必和 MS 完全一致，只要数学等价即可
- **"AG"**（answer given）意味着答案已在题目中给出，你必须推导证明，不能直接用答案来证明自己
- **"AWFW"** 通常对应 $p$ 值和置信区间，因为计算器不同可能有细微差异
- 若题目有 (a)(b)(c) 子问，每个子问独立评分，**不要因为 (a) 错了就放弃 (b)**
:::
