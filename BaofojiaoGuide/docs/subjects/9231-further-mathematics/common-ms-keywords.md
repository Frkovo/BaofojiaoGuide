# Mark Scheme 关键词速查 — Paper 3

## Motion of a projectile

| MS 原句 | 含义 |
|---------|------|
| "**Equations of motion**" (M1) | 写出 $x = Vt\cos\theta$，$y = Vt\sin\theta - \frac12 gt^2$ |
| "**Eliminates $t$**" (M1) | 消去时间得轨迹方程 |
| "**Solves quadratic for $t$**" (M1) | 解关于时间的二次方程 |
| "**Envelope found**" (M1) | 包络线计算 |

## Equilibrium of a rigid body

| MS 原句 | 含义 |
|---------|------|
| "**Resolves forces correctly**" (M1) | 正确的受力分析 |
| "**Takes moments about ...**" (M1) | 对某点取矩 |
| "**Centre of mass found**" (M1) | 质心计算正确 |
| "**Condition for toppling**" (M1) | 倾倒条件分析 |

## Circular motion

| MS 原句 | 含义 |
|---------|------|
| "**Radial acceleration**" (M1) | 向心加速度 $r\omega^2$ 或 $v^2/r$ |
| "**Uses conservation of energy**" (M1) | 能量守恒 |
| "**Reaction = 0**" (M1) | 失去接触条件 $R=0$ |

## Hooke's law

| MS 原句 | 含义 |
|---------|------|
| "**Uses Hooke's law**" (M1) | $T = \lambda x / l$ |
| "**Elastic PE**" (M1) | $\lambda x^2 / (2l)$ |
| "**Conservation of energy**" (M1) | $KE + GPE + EPE$ 守恒 |

## Linear motion under a variable force

| MS 原句 | 含义 |
|---------|------|
| "**Newton's second law**" (M1) | $F = ma$ 写出微分方程 |
| "**Separates variables**" (M1) | 分离变量法 |
| "**Uses $v\frac{dv}{dx}$**" (M1) | 转换为关于位移的形式 |

## Momentum

| MS 原句 | 含义 |
|---------|------|
| "**Conservation of momentum**" (M1) | $m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$ |
| "**Newton's law of restitution**" (M1) | $e = (v_2 - v_1)/(u_1 - u_2)$ |
| "**Oblique impact**" (M1) | 斜碰分解处理 |
