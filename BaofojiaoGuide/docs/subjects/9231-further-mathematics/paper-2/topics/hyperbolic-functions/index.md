# Hyperbolic Functions

## 考纲要求

1. 理解 $\sinh x$、$\cosh x$、$\tanh x$、$\operatorname{sech} x$、$\operatorname{cosech} x$、$\coth x$ 的指数定义：$\sinh x = \frac{e^x - e^{-x}}{2}$，$\cosh x = \frac{e^x + e^{-x}}{2}$
2. 画出双曲函数图像，标出渐近线
3. 证明并运用恒等式：$\cosh^2 x - \sinh^2 x \equiv 1$，$\sinh 2x \equiv 2\sinh x\cosh x$ 等
4. 理解反双曲函数 $\sinh^{-1} x$、$\cosh^{-1} x$、$\tanh^{-1} x$ 的对数形式
5. 推导和使用对数形式

## 常见题型

1. Proving hyperbolic identities
2. Graph sketching with asymptotes
3. Solving hyperbolic equations using exponential form
4. Logarithmic form of inverse hyperbolic functions
5. Integration using hyperbolic substitution

## 核心公式

$$\sinh x = \frac{e^x - e^{-x}}{2},\quad \cosh x = \frac{e^x + e^{-x}}{2}$$
$$\cosh^2 x - \sinh^2 x \equiv 1$$
$$\sinh^{-1} x = \ln(x+\sqrt{x^2+1}),\quad \cosh^{-1} x = \ln(x+\sqrt{x^2-1}),\quad \tanh^{-1} x = \frac12\ln\frac{1+x}{1-x}$$

## 常见错误

- 把 $\cosh^2 x - \sinh^2 x \equiv 1$ 记成 $\cos^2 x + \sin^2 x \equiv 1$ 的形式（错在符号）
- $\cosh^{-1} x$ 定义域忘了：$x \ge 1$；$\tanh^{-1} x$ 定义域忘了：$|x| < 1$
- $\frac{d}{dx}\cosh x = +\sinh x$，不是 $-\sinh x$
