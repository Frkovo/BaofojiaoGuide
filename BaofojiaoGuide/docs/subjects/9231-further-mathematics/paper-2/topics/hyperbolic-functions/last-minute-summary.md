# Last-Minute Summary — Hyperbolic Functions

## Must-know formulas
$\sinh x = \frac{e^x - e^{-x}}{2}$, $\cosh x = \frac{e^x + e^{-x}}{2}$
$\cosh^2 x - \sinh^2 x = 1$
$\sinh 2x = 2\sinh x\cosh x$, $\cosh 2x = 2\cosh^2 x - 1 = 2\sinh^2 x + 1$

## Inverse hyperbolic forms
$\sinh^{-1} x = \ln(x + \sqrt{x^2+1})$
$\cosh^{-1} x = \ln(x + \sqrt{x^2-1})$ $(x \ge 1)$
$\tanh^{-1} x = \frac12\ln\frac{1+x}{1-x}$ $(|x| < 1)$

## Key derivatives
$\frac{d}{dx}\sinh x = \cosh x$, $\frac{d}{dx}\cosh x = \sinh x$ (NO minus sign!)
$\frac{d}{dx}\sinh^{-1}x = \frac{1}{\sqrt{1+x^2}}$

## Red flags
- $\cosh^{-1} x$ requires $x \ge 1$
- $\tanh^{-1} x$ requires $|x| < 1$
- $\frac{d}{dx}\cosh x = +\sinh x$ (NOT $-\sinh x$ like $\cos x$)
