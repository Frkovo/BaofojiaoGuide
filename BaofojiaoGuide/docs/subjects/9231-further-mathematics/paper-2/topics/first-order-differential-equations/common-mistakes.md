# 常见错误 — First Order Differential Equations

## 错误 1：积分因子符号错误

**错误**：未将方程化为标准形式就直接求 $I$。

例如对 $x\frac{dy}{dx} + 2y = e^x$，直接 $P(x) = 2$ 而不是先除以 $x$ 得到 $P(x) = \frac{2}{x}$。

**正解**：先除以 $x$ 的最高幂次，使 $dy/dx$ 系数为 $1$。

## 错误 2：忘记乘积分因子后的 $Q(x)$

**错误**：乘 $I$ 后只写了 $I\frac{dy}{dx} + IPy$，忘记乘 $IQ$ 或写错 $IQ$。

**正解**：明确写出 $I \times \frac{dy}{dx} + I \times P \times y = I \times Q$。

## 错误 3：忽略初值条件

**错误**：通解求完后没有代入 $y(x_0) = y_0$。

**正解**：初值题必须代入求常数，否则即使通解正确也只得到部分分数。

## 错误 4：积分因子化简不到位

**错误**：保留 $e^{\ln(x^2+1)}$ 形式。

**正解**：$e^{\ln(x^2+1)} = x^2 + 1$。

注意 $e^{\frac{1}{2}\ln f(x)} = \sqrt{f(x)}$。

## 错误 5：积分漏常数 $C$

**错误**：积分后写 $yI = \int IQ\,dx$（无 $+C$）。

**正解**：$yI = \int IQ\,dx + C$。漏常数意味失去通解的普遍性。

## 错误 6：$P(x)$ 系数提取疏忽

**错误**：对 $x\frac{dy}{dx} + 2y = e^x$ 直接令 $P(x) = 2$。

**正解**：标准形式要求 $dy/dx$ 系数为 $1$，先除 $x$ 得 $P(x) = 2/x$，$Q(x) = e^x/x$。

## 错误 7：分离变量后积分方向错误

**错误**：$\int y^2\,dy = \int x\,dx$ 写成 $\int \frac{1}{y^2}\,dy = \int x\,dx$。

**正解**：仔细移项，确保 $y$ 项和 $dy$ 在同侧，$x$ 项和 $dx$ 在同侧。

## 错误 8：特解形式未最简化

**错误**：最终答案保留 $y = \frac{x^4/4 + C}{x}$。

**正解**：应化简为 $y = \frac{x^3}{4} + \frac{C}{x}$，再代入初值得具体数值。
