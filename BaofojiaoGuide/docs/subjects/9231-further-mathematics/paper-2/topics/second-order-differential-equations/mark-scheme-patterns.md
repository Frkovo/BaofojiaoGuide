# Mark Scheme Patterns — Second Order Differential Equations

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 辅助方程正确 | **M1** | $am^2 + bm + c = 0$ |
| 特征根正确 | **A1** | 包括实根、重根、复根 |
| CF 形式正确 | **A1** | 与根的类型对应 |
| PI 试设合理 | **M1** | 与 RHS 形式匹配 |
| 代入比较系数 | **M1** | 代入方程化简 |
| 系数值正确 | **A1** | $\alpha, \beta, \gamma$ 等 |
| PI 正确写出 | **A1** | 完整形式 |
| 通解 = CF + PI | **A1** | 合并 |
| 初值代入 | **M1** | 使用 $y(x_0)$ 和 $y'(x_0)$ |
| 常数正确 | **A1** | $A$ 和 $B$ 的值 |

## 常见给分模式

### 常系数题型（6-11 分）

**M1** (AE) → **A1** (根) → **A1** (CF) → **M1** (PI 试设) → **M1** (代入比较) → **A1** (系数) → **A1** (PI) → **A1** (通解) → 如需初值则加 **M1 A1**

### 6 分小题模式

- AE + 根：2 分
- CF：1 分
- PI 过程：2 分
- 最终通解：1 分

### Euler-Cauchy 题型（4+7 分）

- (a) 变换 + 推导 4 分：**M1 M1 A1 A1**
- (b) 求解 7 分：AE **M1** → 根 **A1** → CF **A1** → PI **M1 A1** → 通解 **A1** → 代回 **A1**

### 11 分大题的常见分布

- 齐次部分（AE + CF）：3 分
- 特解 PI：5 分
- 通解：1 分
- 初值条件：2 分

## 评分陷阱

- PI 与 CF 冲突未乘 $x$ 通常扣 3 分以上（**M1** + **A1** + **A1**）
- CF 写错常数形式（如重根漏 $x$）扣 **A1**
- 代入初值时，$y'(x_0)$ 忘记先对通解求导扣 **M1**
