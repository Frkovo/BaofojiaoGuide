# Implicit Differentiation

## 考纲要求

- 对隐函数方程求一阶导数 $\frac{dy}{dx}$
- 对隐函数方程求二阶导数 $\frac{d^2y}{dx^2}$
- 在给定点处求导数值
- 结合切线法线方程综合考查

## 常见题型

| 题型 | 分值 | 频率 |
|------|------|------|
| 求一阶导数 $\frac{dy}{dx}$ | 3 marks | 必考 |
| 求二阶导数 $\frac{d^2y}{dx^2}$ | 5 marks | 高频 |
| 特定点处的导数值（含切线） | 8 marks | 中频 |

## 核心公式

隐函数求导法则：

$$\frac{d}{dx}f(y)=f'(y)\frac{dy}{dx}$$

常用乘积法则：

$$\frac{d}{dx}(xy)=y+x\frac{dy}{dx}$$

二阶导数：

$$\frac{d^2y}{dx^2}=\frac{d}{dx}\left(\frac{dy}{dx}\right)=\frac{d}{dy}\left(\frac{dy}{dx}\right)\cdot\frac{dy}{dx}$$

## 常见错误

1. **忘记乘 $\frac{dy}{dx}$**：对 $y$ 的函数求导时忘记链式法则
2. **乘积法则遗漏**：处理 $xy$、$x^2y$ 等项时漏用乘积法则
3. **代入顺序错误**：求出 $\frac{dy}{dx}$ 表达式后未代入具体点坐标
4. **二阶导数公式混淆**：误用商的导数规则而非隐函数方法
5. **化简遗漏**：最终表达式未按要求化简为最简形式
