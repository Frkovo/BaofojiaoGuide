# Last-Minute Summary — Matrices (FP2)

## Must-know
- Characteristic equation: det(A - λI) = 0
- Eigenvectors: (A - λI)e = 0
- Diagonalisation: A = PDP⁻¹ with D diagonal of eigenvalues, P columns are eigenvectors
- Powers: Aⁿ = PDⁿP⁻¹

## Key warnings
- det A = 0 means NO unique solution (could be infinite or none — check consistency!)
- Eigenvector order must match eigenvalue order in D
- Cayley–Hamilton: A satisfies its own characteristic equation → can find A⁻¹ without determinant

## Common question types
1. Solve 3×3 system + geometric interpretation
2. Find eigenvalues/vectors
3. Diagonalise + find Aⁿ
4. Use Cayley–Hamilton for inverse or powers
