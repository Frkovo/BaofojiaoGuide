# Matrices 评分标准规律

---

## 题型 1：特征值与特征向量（2–5 分）

| 步骤 | 分值 | 评分标准 |
|------|------|----------|
| 写出 $\det(A - \lambda I) = 0$ | **M1** | 方法分：正确设出行列式 |
| 展开行列式 | **M1** | 方法分：正确展开（注意符号） |
| 解特征多项式 | **A1** | 答案分：特征值正确 |
| 对每个特征值列出齐次方程组 | **M1** | 方法分：代入 $\lambda$ |
| 解出特征向量 | **A1** | 答案分：特征向量正确（非零） |

**典型扣分点**：
- 展开行列式符号错误 → 后续全错，仅得展开前的 **M1**
- 特征向量未写为列向量形式 → 扣 **A1**

---

## 题型 2：Cayley-Hamilton 定理（3–6 分）

| 步骤 | 分值 | 评分标准 |
|------|------|----------|
| 写出特征方程 | **M1** | 方法分 |
| 代入 $A$ 得矩阵方程 | **A1** | 答案分：方程正确 |
| 求 $A^{-1}$：整理表达式 | **M1** | 方法分：正确移项 |
| 最终 $A^{-1}$ 正确 | **A1** | 答案分 |
| 求 $(A^{-1})^2$ 或其他变形 | **M1 + A1** | 额外步骤分 |

**典型模式**：
- s20/21 Q8(b)：特征方程 + $A^{-1}$ → **M1, A1, M1, A1**
- s21/23 Q7：特征方程 + 高次幂 → **M1 A1, M1 A1**
- s25/21 Q8(d)：化简 $(A-2I)^3$ → **M1, M1, A1**

---

## 题型 3：对角化 / $A^n = PDP^{-1}$（5–7 分）

| 步骤 | 分值 | 评分标准 |
|------|------|----------|
| 求出一个特征值 | **B1** | 已知或简单计算得出 |
| 求出全部特征值 | **B1** | 所有特征值正确 |
| 求特征向量（方法） | **M1** | 方法分 |
| 特征向量（答案） | **A1** | 所有特征向量正确 |
| 构造 $P$ 和 $D$ | **A1** | $P$ 和 $D$ 正确 |
| 指出 $A^n = PD^nP^{-1}$ 或写出 $D^n$ | **B1** | 理解关系 |
| 计算 $P^{-1}$ | **M1** | 方法分 |
| $P^{-1}$ 正确 | **A1** | 答案分 |

**典型模式**：
- s20/21 Q8(c)：$A^5 = PDP^{-1}$，仅要求 $P$ 和 $D$（不含 $P^{-1}$）→ **7 分**
- s24/21 Q9：$(14A+24I)^2 = PDP^{-1}$ → **7 分**

---

## 常见评分规则

1. **错误累加**：前一步错导致后一步错，后一步不得分（除非问题独立）
2. **方法分**（**M1**）：即使计算错误，正确的方法仍给方法分
3. **答案分**（**A1**）：必须在方法正确的前提下给出正确答案
4. **独立部分**：对角化中的 $P$ 和 $D$ 通常各为 **A1**，互不影响
5. **验证要求**：有时要求验证 $A = PDP^{-1}$，验证步骤为 **M1**
