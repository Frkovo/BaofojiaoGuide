# Systems of Linear Equations 评分标准规律

---

## 题型 1：行列式条件（2–5 分）

| 步骤 | 分值 | 评分标准 |
|------|------|----------|
| 写出系数矩阵 | **B1** | 或直接写行列式 |
| 正确写出 $\det A$ | **M1** | 方法分 |
| 展开行列式 | **M1** | 方法分：展开过程正确 |
| 化简得到参数方程 | **A1** | 答案分 |
| 写出参数条件 | **A1** | 答案分：如 $a = \frac{3}{2}$ 或 $a \neq -\frac{1}{2}$ |

**典型模式**：
- s20/21 Q8(a)：求 $a$ 使得无唯一解（齐次）→ **M1, M1, A1**
- s21/21 Q1(a)：求 $a$ 使得有唯一解 → **M1, M1, A1, A1**
- w20/21 Q3(a)：证明无唯一解 → **M1, A1**

---

## 题型 2：相容性判断（3–4 分）

| 步骤 | 分值 | 评分标准 |
|------|------|----------|
| 写出增广矩阵 | **M1** | 方法分 |
| 行变换 | **M1** | 方法分：至少一步变换正确 |
| 得到参数条件 | **M1** | 方法分：从变换得出条件 |
| 判断相容/矛盾 | **A1** | 答案分 |

**典型模式**：
- w23/21 Q1：矛盾参数值 → **M1, M1, M1, A1**
- w22/22 Q1(b)：判断相容性 → **M1, A1**

**注意**：当问题分两问时，第一问唯一解往往只需要 $\det A = 0$，第二问增广矩阵需要进一步判断相容性。

---

## 题型 3：几何解释（2–3 分）

| 几何情形 | 分值 | 关键描述词 |
|----------|------|------------|
| 唯一解 → 三平面交于一点 | **B1** | "meet at a point" |
| 无穷多解 → 三平面交于一条直线 | **B1** | "meet in a line" / "common line" |
| 矛盾无解 → 三棱柱 | **B1** | "triangular prism" / "no common point" |
| 矛盾无解 → 两平面平行 | **B1** | "two planes are parallel" |

**典型模式**：
- w20/21 Q3(b)：$k = -4$ 相容，无穷多解 → **B1** 相容 + **B1** 交于一条直线 + **B1** 三平面
- w20/21 Q3(c)：$k = -2$ 矛盾 → **B1** 矛盾 + **B1** 三棱柱
- w20/21 Q3(d)：三棱柱解释 → **B1** 两两相交 + **B1** 交线平行

---

## 常见评分规则

1. **行列式计算**：$\det A$ 展开过程即使最后化简出错，仍给 **M1**
2. **行变换**：每一步正确的行变换给 **M1**（可给多个）
3. **几何描述**：只用关键词即可得分，不需要长篇解释
4. **参数表达**：参数化解中参数符号是否使用 $t$ 不重要，但必须指明参数
