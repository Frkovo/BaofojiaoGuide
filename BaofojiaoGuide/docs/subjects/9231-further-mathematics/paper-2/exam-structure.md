# 考试结构 — Paper 2

## 试卷概况

| 项目 | 详情 |
|------|------|
| 试卷代码 | 9231/22 |
| 名称 | Further Pure Mathematics 2 |
| 时长 | 2 小时（120 分钟） |
| 满分 | 75 |
| 题量 | 9–11 题，全部必答 |
| 计算器 | 标准科学计算器（禁止 CAS/图形） |
| 公式表 | MF19 考场提供 |

## 答题规范

- **在试卷上作答**，展示所有步骤
- **非精确答案**：保留 3 位有效数字（角度保留 1 位小数）
- **优先保留精确形式**：$\sqrt{3}$、$\pi/6$，不要写小数
- 用黑色/深蓝色笔，铅笔仅限画图

## Command Words（指令词）

| 指令词 | 要求 |
|--------|------|
| **Show that** | 完整推导，每步有依据 |
| **Find / Determine** | 求出答案，展示关键步骤 |
| **Solve** | 求所有解（可能包含复数） |
| **Prove** | 逻辑清晰的论证 |
| **Sketch** | 展示关键特征：渐近线、截距、极值点 |
| **Hence** | **必须**使用上一步的结论 |
| **State** | 不需要过程，但答案必须精确 |
