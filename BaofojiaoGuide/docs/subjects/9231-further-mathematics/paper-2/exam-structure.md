# 考试结构

## 试卷概况

| 项目 | 细节 |
|------|------|
| **考试时长** | 2 小时 |
| **总分** | 75 |
| **题目数量** | 8 道必答题 |
| **每题最高分** | 10-15 分（最后一题通常最重） |
| **答题方式** | 试卷上直接作答 |
| **计算器** | 科学计算器，不得使用图形计算器 |
| **公式表** | List of Formulae (MF19) |

## 典型分值分布

| 题目 | 典型 Topics | 典型分值 | 难度 |
|------|-------------|---------|------|
| Q1 | 一阶 ODE / 复根 / Maclaurin 级数 | 5-7 | 易 |
| Q2 | Maclaurin 级数 / 递推公式 / 二阶 ODE | 5-7 | 易 |
| Q3 | 复数 / 线性方程组 / 隐式微分 | 7-9 | 中 |
| Q4 | Riemann 求和 / 矩阵 / 参数微分 | 8-10 | 中 |
| Q5 | 双曲函数 / 弧长 / 二阶 ODE | 8-11 | 中 |
| Q6 | De Moivre / 递推公式 / 积分 | 10-11 | 中-难 |
| Q7 | ODE（一阶/二阶）/ 矩阵 | 11 | 难 |
| Q8 | 矩阵对角化 / 双曲函数 / 复数求和 | 14-15 | 难 |

## 指令词表

| 指令词 | 中文要求 |
|--------|---------|
| **Show that** | 证明给定结果，需要展示完整推导过程 |
| **Find** | 求解得到最终答案 |
| **Sketch** | 画出草图，标注关键点和渐近线 |
| **Determine** | 确定，可能需要判断或计算 |
| **Express** | 用指定形式表达 |
| **Hence** | 利用前一部分的结果 |
| **State** | 直接写出，不需推导 |
| **Deduce** | 推导出，基于已有结果 |
| **Interpret geometrically** | 给出几何解释 |
| **Solve** | 解方程 |
| **Differentiate** | 求导 |
| **Integrate** | 求积分 |

## 答题规范

1. 所有必要步骤必须清晰展示
2. 无过程的正确答案可能不得分
3. 除非题中指定，答案可用分数、小数或标准形式
4. 数值答案保留 3 位有效数字，角度保留 1 位小数
5. 使用黑色或深蓝色笔，图表可用 HB 铅笔
6. 禁止使用可擦笔或修正液
