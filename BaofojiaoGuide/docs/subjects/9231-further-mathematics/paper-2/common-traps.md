# Paper 2 常见陷阱

## Trap 1: "Show that" 只写结果不写过程
**问题：** MS 给的是步骤分（M marks），光写答案得 0 分。
**对策：** **每一步代数都写出来。**

## Trap 2: 不定积分忘了 $+C$
**问题：** 不定积分没有常数项。
**对策：** 永远加 $+C$，微分方程里它变成 CF 的一部分。

## Trap 3: 精确值写成小数
**问题：** 答案 $2+\sqrt{3}$ 写成 3.73 会扣 A mark。
**对策：** 除非题目明确要求 3 s.f.，否则保留精确形式。

## Trap 4: 反双曲函数的定义域
- $\cosh^{-1} x$ 要求 $x \ge 1$
- $\tanh^{-1} x$ 要求 $|x| < 1$
- $\sin^{-1} x$ 要求 $|x| \le 1$

## Trap 5: 矩阵乘法顺序
$AB$ 表示"先做 $B$ 再做 $A$"，顺序不能交换。

## Trap 6: 特征方程符号
统一用 $\det(A - \lambda I) = 0$，别用 $\det(\lambda I - A)$ 搞混符号。

## Trap 7: Maclaurin 展开点
Maclaurin 是在 $x=0$ 展开。如果题目问别的点（很少）需要用 Taylor 级数。

## Trap 8: 代换后忘了换积分限
设 $u = g(x)$ 后，$x=a$ 变成 $u=g(a)$，$x=b$ 变成 $u=g(b)$。

## Trap 9: CF 和 PI 重叠
右端项已经在 CF 里了？特解要乘以 $x$（重根乘 $x^2$）。

## Trap 10: "Hence" 必须用上一步结论
(b) 说 "hence" 就必须用 (a) 的结论，自己另起炉灶不得分。
