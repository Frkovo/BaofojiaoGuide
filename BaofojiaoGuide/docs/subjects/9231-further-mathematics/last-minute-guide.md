# 考前速通指南 — Paper 3

## 必须记住的公式（MF19 不一定全有）

### Projectile Motion
$$x = Vt\cos\theta,\qquad y = Vt\sin\theta - \frac12 gt^2$$

### Circular Motion
$$F = mr\omega^2 = \frac{mv^2}{r}$$

### Hooke's Law
$$T = \frac{\lambda}{l}x,\qquad \text{EPE} = \frac{\lambda x^2}{2l}$$

### Momentum & Restitution
$$m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$$
$$e = \frac{v_2 - v_1}{u_1 - u_2}$$

## 见到什么就先做什么

| 看到这个… | 第一步 |
|-----------|--------|
| Projectile | 写出 $x = Vt\cos\theta$，$y = Vt\sin\theta - \frac12 gt^2$ |
| Rigid body equilibrium | 画受力图，取矩 |
| Circular motion | 确定向心力来源 |
| Elastic string/spring | 算伸长量 $x$ |
| Variable force | 牛顿第二定律 $\to$ 微分方程 |
| Collision | 动量守恒 $+$ NEL |
