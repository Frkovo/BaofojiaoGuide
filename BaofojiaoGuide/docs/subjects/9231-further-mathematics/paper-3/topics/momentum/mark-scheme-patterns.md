# Mark Scheme Patterns — Momentum

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 确定正方向 | **B1** | 明确正方向 |
| 写出动量守恒 | **M1** | $m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$ |
| 动量守恒方程正确 | **A1** | 代入正确值 |
| 写出 NEL | **M1** | $e = \frac{v_2 - v_1}{u_1 - u_2}$ |
| NEL 方程正确 | **A1** | 分子分母顺序正确 |
| 解方程组 | **M1** | 联立求解 |
| 最终速度值 | **A1** | 含方向（±） |
| 斜碰法线分解 | **M1** | 沿法线/切线 |
| 切线速度不变 | **B1** | 识别 |
| 动能损失计算 | **M1** | $\frac{1}{2}mu^2 - \frac{1}{2}mv^2$ |
| 最终 KE 损失 | **A1** | 正值 |

## 常见给分模式

- 对心碰撞：正方向 **B1** → 动量守恒 **M1 A1** → NEL **M1 A1** → 解 **M1** → 速度 **A1**
- 斜碰：分解 **M1** → 法线动量 **M1** → NEL **M1** → 切向不变 **B1** → 合成 **M1 A1**
- 连续碰撞：逐对处理，每对仿照对心碰撞模式
