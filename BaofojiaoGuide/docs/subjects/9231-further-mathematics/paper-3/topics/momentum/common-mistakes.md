# 常见错误 — Momentum

## 错误 1：动量守恒方向错误

**错误**：将所有速度视为正数代入。

**正解**：先确定正方向，相反方向的速度为负值。

## 错误 2：NEL 公式顺序

**错误**：分子分母顺序颠倒。

**正解**：$e = \frac{v_2 - v_1}{u_1 - u_2}$，注意分子是碰后速度差，分母是碰前速度差。

## 错误 3：斜碰只用一个方程

**错误**：斜碰只列动量守恒。

**正解**：沿法线方向用动量守恒 + NEL，切线方向速度不变。

## 错误 4：完全弹性时 $e = 0$

**错误**：认为完全弹性和完全非弹性混淆。

**正解**：完全弹性 $e = 1$，完全非弹性 $e = 0$。

## 错误 5：动能损失计算

**错误**：只算了初动能或只算了末动能。

**正解**：动能损失 $=$ 初动能 $-$ 末动能，必须是正值。
