# Momentum（动量）

## 考纲要求

- 动量守恒定律
- Newton 实验公式（恢复系数）$e = \frac{v_2 - v_1}{u_1 - u_2}$
- 对心碰撞（direct impact）
- 斜碰（oblique impact）沿法线方向处理
- 碰撞中的动能损失
- 连续碰撞（多粒子系统）

## 核心公式

| 公式 | 说明 |
|------|------|
| $m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$ | 动量守恒 |
| $e = \frac{v_2 - v_1}{u_1 - u_2}$ | Newton 实验公式 |
| $0 \leq e \leq 1$ | 恢复系数范围 |
| $\Delta KE = \frac{1}{2}m_1(u_1^2 - v_1^2) + \frac{1}{2}m_2(u_2^2 - v_2^2)$ | 动能损失 |
| 斜碰：切线方向速度不变 | 分量形式 |

## 常见问题类型

1. 对心碰撞求末速度
2. 斜碰分解处理
3. 与固定面碰撞
4. 连续碰撞
5. 计算动能损失

## 解题步骤

1. 明确系统，确定正方向
2. 写出碰撞前后总动量
3. 写出 Newton 实验公式
4. 解方程组
5. （可选）计算动能损失
