# Syllabus Points — Momentum

## 核心知识点

### 1. 动量与冲量

- 动量 $\mathbf{p} = m\mathbf{v}$
- 冲量 $\mathbf{I} = \mathbf{F}\Delta t = \Delta\mathbf{p}$

### 2. 动量守恒定律

- $m_1\mathbf{u}_1 + m_2\mathbf{u}_2 = m_1\mathbf{v}_1 + m_2\mathbf{v}_2$
- 前提：系统不受外力或外力合力为零
- 适用于沿同一直线或分量形式

### 3. Newton 实验公式

- $e = \frac{v_2 - v_1}{u_1 - u_2}$
- $e$ 为恢复系数，$0 \leq e \leq 1$
- $e = 0$：完全非弹性碰撞（碰后粘合）
- $e = 1$：完全弹性碰撞（动能无损失）

### 4. 对心碰撞与斜碰

- 对心碰撞：两球质心连线与速度方向共线
- 斜碰：
  - 沿法线方向（line of centres）：动量守恒 + NEL
  - 沿切线方向：各球速度分量不变

### 5. 与固定面碰撞

- 碰后速度 = $-e \times$ 碰前速度（沿法线方向）
- 动能损失计算

### 6. 连续碰撞

- 逐对处理，先碰的先解
- 每次碰撞用动量守恒 + NEL
- 注意区分碰前和碰后状态
