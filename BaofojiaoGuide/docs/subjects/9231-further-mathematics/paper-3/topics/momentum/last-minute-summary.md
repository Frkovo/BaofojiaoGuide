# 考前速记 — Momentum

## 核心公式

| 公式 | 说明 |
|------|------|
| $m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$ | 动量守恒 |
| $e = \frac{v_2 - v_1}{u_1 - u_2}$ | Newton 实验公式 |
| $0 \leq e \leq 1$ | 恢复系数范围 |
| $\Delta KE = \sum\frac{1}{2}mu^2 - \sum\frac{1}{2}mv^2$ | 动能损失 |

## 碰撞类型

| 类型 | $e$ 值 | 特点 |
|------|--------|------|
| 完全弹性 | $e = 1$ | 动能守恒 |
| 完全非弹性 | $e = 0$ | 碰后粘合 |
| 非完全弹性 | $0 < e < 1$ | 动能部分损失 |

## 易错点

- 先定正方向
- NEL 分子分母顺序勿颠倒
- 斜碰分法线和切线处理
- 连续碰撞逐对求解
- 动能损失一定是正值

## 解题策略

1. 定正方向
2. 动量守恒
3. NEL
4. 解方程组
5. （可选）动能损失
