# Hooke's Law（胡克定律）

## 考纲要求

- 使用 $T = \frac{\lambda x}{L}$ 计算弹性绳/弹簧的张力
- 计算弹性势能 $\text{EPE} = \frac{\lambda x^2}{2L}$
- 分析包含弹性绳的系统能量
- 处理弹性绳与圆周运动、斜面等组合问题

## 核心公式

| 公式 | 说明 |
|------|------|
| $T = \frac{\lambda x}{L}$ | 张力（$\lambda$ 为弹性模量，$L$ 为自然长度） |
| $\text{EPE} = \frac{\lambda x^2}{2L}$ | 弹性势能 |
| $T = kx$ | 等价形式（$k = \lambda / L$） |

## 常见问题类型

1. 弹性绳静态平衡（求伸长量）
2. 粒子沿斜面被弹性绳拉动
3. 弹性绳在圆周运动中的应用
4. 能量法求速度或位移

## 解题步骤

1. 确定自然长度 $L$ 和实际长度
2. 计算伸长量 $x$
3. 写出张力
4. 列平衡方程或能量方程
5. 注意弹性绳不能受压
