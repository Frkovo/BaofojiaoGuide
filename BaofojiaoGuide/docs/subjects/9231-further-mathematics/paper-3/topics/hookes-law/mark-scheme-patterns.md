# Mark Scheme Patterns — Hooke's Law

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 识别伸长量 $x$ | **M1** | 实际长度 − 自然长度 |
| 正确使用胡克定律 | **M1** | $T = \frac{\lambda x}{L}$ |
| 平衡方程 | **M1** | 如 $T = mg$ |
| 代入解 $x$ | **M1** | 正确代数 |
| 弹性势能公式 | **M1** | $\text{EPE} = \frac{\lambda x^2}{2L}$ |
| 能量守恒方程 | **M1** | KE + GPE + EPE |
| 中间值 | **A1** | 部分正确 |
| 最终数值 | **A1** | 3 s.f. |
| 弹性绳松弛条件 | **B1** | 识别 $T = 0$ 或 EPE = 0 |

## 常见给分模式

- 静态平衡：识别力 **B1** → 平衡方程 **M1** → 胡克定律 **M1** → $x$ **A1**
- 能量法：初末能量 **M1** → EPE 正确 **A1** → 代入已知 **M1** → 解 **A1**
- 圆周+弹性绳：向心力 **M1** → 胡克定律 **M1** → 几何 **M1** → 解 **A1**
