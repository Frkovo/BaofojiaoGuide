# Syllabus Points — Hooke's Law

## 核心知识点

### 1. 胡克定律

- $T = \frac{\lambda x}{L}$，其中 $\lambda$ 为弹性模量，$L$ 为自然长度，$x$ 为伸长量
- 等价形式 $T = kx$，$k = \lambda / L$ 为劲度系数
- 弹性绳仅能承受拉力，不能承受压力

### 2. 弹性势能

- $\text{EPE} = \frac{\lambda x^2}{2L}$
- 弹性势能是 $\frac{1}{2}kx^2$ 的推广形式
- 能量守恒：$\Delta KE + \Delta GPE + \Delta EPE = 0$（无外力做功）

### 3. 弹性绳的组合

- 多根弹性绳并联等效 $\lambda$ 相加
- 串联时等效 $\lambda$ 满足 $\frac{1}{\lambda_{\text{eq}}} = \frac{1}{\lambda_1} + \frac{1}{\lambda_2}$

### 4. 弹性力学的其他应用

- 弹性绳 + 圆周运动
- 弹性绳 + 斜面
- 弹性绳在 SHM 中的角色
