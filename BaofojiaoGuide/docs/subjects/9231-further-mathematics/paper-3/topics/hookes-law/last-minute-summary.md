# 考前速记 — Hooke's Law

## 核心公式

| 公式 | 说明 |
|------|------|
| $T = \frac{\lambda x}{L}$ | 胡克定律 |
| $\text{EPE} = \frac{\lambda x^2}{2L}$ | 弹性势能 |
| $k = \frac{\lambda}{L}$ | 等效劲度系数 |

## 易错点

- $L$ 是自然长度（不是总长度）
- $x$ 是伸长量（总长 − 自然长）
- 弹性绳不能受压（$x \leq 0$ 时 $T = 0$）
- 能量法别忘了 EPE

## 解题策略

1. 确定自然长度 $L$ 和伸长量 $x$
2. 写出 $T = \lambda x / L$
3. 列平衡方程或能量守恒
4. 注意松弛条件
