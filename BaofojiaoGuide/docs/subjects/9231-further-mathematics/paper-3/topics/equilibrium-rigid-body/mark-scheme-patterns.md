# Mark Scheme Patterns — Equilibrium of Rigid Body

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 画受力图（认出力） | **B1** | 识别所有外力 |
| 取力矩方程 | **M1** | 选择正确支点 |
| 力臂正确 | **A1** | 垂直距离 |
| 水平/竖直合力方程 | **M1** | 各 1 分 |
| 解方程组 | **M1** | 消元法 |
| 中间值 | **A1** | 部分正确 |
| 最终答案 | **A1** | 3 s.f. |
| 质心加权平均公式 | **M1** | 正确应用 |
| 倾倒判断 | **M1 A1** | 比较力矩/位置 |

## 常见给分模式

- 找质心问题：分面积 **B1** → 各质心位置 **M1** → 加权平均 **M1** → 答案 **A1**
- 梯子问题：力矩方程 **M1** → 解反力 **A1** → 摩擦条件 **M1** → $\mu$ **A1**
- 倾倒分析：滑动力 **M1** → 倾倒力矩 **M1** → 比较 **A1**
