# Solution Methods — Equilibrium of Rigid Body

## 方法一：标准平衡法

适用于均匀杆、简单支撑问题。

:::info[Steps]

1. 画受力图，标示所有外力和支点位置
2. 选择支点（通常选未知力最多的点取矩）
3. $\sum F_x = 0$，$\sum F_y = 0$
4. $\sum M = 0$（对所选支点）
5. 联立求解

:::

## 方法二：质心法

适用于复合体找质心。

:::info[Steps]

1. 将物体分解为规则形状
2. 标出各部分质心位置
3. 使用加权平均公式：
   $\bar{x} = \frac{m_1x_1 + m_2x_2 + \dots}{m_1 + m_2 + \dots}$
4. 对 $y$ 坐标同理

:::

## 方法三：倾倒分析

适用于判断物体是否倾倒。

:::info[Steps]

1. 找质心位置
2. 确定支点（接触面边缘）
3. 过质心画竖直线
4. 若竖直线超出支点范围则倾倒
5. 临界状态：竖直线过支点

:::

## 方法四：梯子问题

适用于梯子靠墙的平衡。

:::info[Steps]

1. 梯子与墙、地接触点有反力
2. 地面有摩擦力 $F \leq \mu R$
3. 对任意点取矩（通常取梯子与地接触点）
4. 消去未知反力
5. 代入摩擦条件求临界值

:::
