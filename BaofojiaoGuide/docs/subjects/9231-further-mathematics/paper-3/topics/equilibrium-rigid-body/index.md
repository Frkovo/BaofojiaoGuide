# Equilibrium of Rigid Body（刚体平衡）

## 考纲要求

- 计算力矩（扭矩）
- 刚体平衡条件：合力为零、合力矩为零
- 确定复合体的质心位置
- 分析倾倒（toppling）和滑动（sliding）条件
- 处理涉及摩擦的平衡问题（如梯子问题）

## 核心公式

| 公式 | 说明 |
|------|------|
| $\mathbf{M} = \mathbf{r} \times \mathbf{F}$ | 力矩（向量形式） |
| $M = Fd$ | 力矩大小（$d$ 为垂直距离） |
| $\sum F_x = 0,\ \sum F_y = 0$ | 合力为零 |
| $\sum M = 0$ | 对任意点合力矩为零 |
| $\bar{x} = \frac{\sum m_i x_i}{\sum m_i}$ | 质心 $x$ 坐标 |
| $\bar{y} = \frac{\sum m_i y_i}{\sum m_i}$ | 质心 $y$ 坐标 |
| $F_{\max} = \mu R$ | 最大静摩擦力 |

## 常见问题类型

1. 均匀杆受力平衡
2. 复合体找质心
3. 梯子平衡问题
4. 倾倒与滑动判断

## 解题步骤

1. 画清晰的受力图
2. 将力分解为水平和竖直分量
3. 选择方便的支点取力矩
4. 列出三个平衡方程
5. 解方程组
