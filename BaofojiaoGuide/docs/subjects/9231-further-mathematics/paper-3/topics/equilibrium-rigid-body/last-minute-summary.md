# 考前速记 — Equilibrium of Rigid Body

## 核心公式

| 公式 | 说明 |
|------|------|
| $M = Fd$ | 力矩（$d$ 是垂直距离） |
| $\sum F_x = 0$ | 水平合力为零 |
| $\sum F_y = 0$ | 竖直合力为零 |
| $\sum M = 0$ | 合力矩为零 |
| $\bar{x} = \frac{\sum m_i x_i}{\sum m_i}$ | 质心 $x$ 坐标 |
| $F_{\max} = \mu R$ | 最大静摩擦力 |

## 易错点

- 力矩中 $d$ 是垂直距离（不是斜边）
- 复合体质心用加权平均
- 倾倒条件：重心竖直线超出支点范围
- 梯子：墙通常光滑，地粗糙

## 解题策略

1. 画受力图
2. 选择合适支点取矩（消去未知反力）
3. 列三个平衡方程
4. 联立求解
