# Mark Scheme Patterns — Linear Motion Under Variable Force

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 写出 $F = ma$ | **M1** | 正确列出方程 |
| 选择合适形式 | **M1** | $dv/dt$ 或 $v dv/dx$ |
| 分离变量 | **M1** | 正确分离 |
| 积分 | **M1** | 正确积分公式 |
| 积分常数 | **M1** | 代入初始条件 |
| 中间表达式 | **A1** | 部分正确 |
| 最终答案 | **A1** | 3 s.f. |
| 极限速度 | **B1** | 识别 $a = 0$ 时的速度 |
| 换元技巧 | **M1** | 如部分分式或三角换元 |

## 常见给分模式

- 求 $v(t)$：$F = m dv/dt$ **M1** → 分离变量 **M1** → 积分 **M1** → 初始条件 **M1** → $v(t)$ **A1**
- 求 $v(x)$：$v dv/dx$ **M1** → 分离 **M1** → 积分 **M1** → $v(x)$ **A1**
- 空气阻力 $kv$：同 $v(t)$ 模式 + 极限速度 **B1**
