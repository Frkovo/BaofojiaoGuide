# 常见错误 — Linear Motion Under Variable Force

## 错误 1：积分后忘记常数

**错误**：积分结果不加常数 $C$。

**正解**：不定积分后必须加积分常数，代入初始条件求解。

## 错误 2：$a = v\frac{dv}{dx}$ 不会用

**错误**：遇到求 $v$ 关于 $x$ 的问题不知道换形式。

**正解**：当问题涉及位移和速度关系时，用 $a = v\frac{dv}{dx}$。

## 错误 3：空气阻力方向

**错误**：阻力方向与运动方向相同。

**正解**：阻力总是与运动方向相反，注意正负号。

## 错误 4：分离变量不彻底

**错误**：分离变量时漏因子。

**正解**：将所有含 $v$ 的项移到一边，所有含 $t$ 或 $x$ 的项移到另一边。

## 错误 5：极限速度推导

**错误**：直接设 $v = \infty$。

**正解**：极限速度时 $a = 0$，即 $mg = kv$ 或 $mg = kv^2$。
