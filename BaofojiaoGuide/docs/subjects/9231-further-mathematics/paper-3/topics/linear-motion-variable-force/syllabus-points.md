# Syllabus Points — Linear Motion Under Variable Force

## 核心知识点

### 1. Newton 第二定律的微分形式

- $F = ma = m\frac{dv}{dt}$
- $F = m\frac{d^2x}{dt^2}$
- $a = v\frac{dv}{dx}$

### 2. 常见的力模型

- 阻力正比于速度：$\text{阻力} = kv$
- 阻力正比于速度平方：$\text{阻力} = kv^2$
- 回复力 $F = -kx$（简谐运动前兆）
- 与位置有关的力 $F = F(x)$
- 与速度有关的力 $F = F(v)$

### 3. 微分方程求解方法

- 分离变量法
- 一阶线性微分方程
- 初始条件确定积分常数

### 4. 常见求解目标

- 速度作为时间函数 $v(t)$
- 位移作为时间函数 $x(t)$
- 速度作为位移函数 $v(x)$
- 极限速度（terminal velocity）
