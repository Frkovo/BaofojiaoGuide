# Mark Scheme Patterns — Projectile Motion

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 分解初速度 | **M1** | $u_x = u\cos\theta$，$u_y = u\sin\theta$ |
| 水平/竖直分量正确 | **A1** | 每个分量各 1 分 |
| 写出运动方程 | **M1** | 使用正确的 suvat 公式 |
| 代入已知值 | **M1** | 正确代入数值 |
| 中途结果 | **A1** | 部分正确可得 |
| 最终答案 | **A1** | 需 3 s.f. |
| 正确单位 | **B1** | m/s, m, s 等 |
| 轨迹方程证明 | **M1 A1** | 完整推导过程 |
| 斜面问题中分解 $g$ | **M1** | 沿斜面和垂直斜面 |

## 常见给分模式

- **(i)** 通常是直接计算题：**M1 A1** 或 **B1**（2 分）
- **Show that** 题：多步 **M1** + final **A1**
- 轨迹方程相关问题：至少 3 个 **M1**
- 反推初值：列方程 **M1** → 求解 **M1** → 答案 **A1**
