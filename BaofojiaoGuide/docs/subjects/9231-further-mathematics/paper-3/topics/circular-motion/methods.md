# Solution Methods — Circular Motion

## 方法一：水平圆周运动

适用于圆锥摆、倾斜轨道。

:::info[Steps]

1. 确定圆周半径 $r$
   - 圆锥摆：$r = L\sin\theta$
   - 轨道：$r$ 已知
2. 列出向心力方程 $F = m\frac{v^2}{r} = mr\omega^2$
3. 竖直方向平衡（或合力提供向心力分量）
4. 联立求解

:::

## 方法二：竖直圆周运动

适用于粒子在竖直圆环或摆线中运动。

:::info[Steps]

1. 确定所给点的位置（角度或高度）
2. 用能量守恒求该点速度：
   $\frac{1}{2}mv_A^2 + mgh_A = \frac{1}{2}mv_B^2 + mgh_B$
3. 列向心力方程：向心力 $=$ 各力径向分量之和
4. 求解张力或反力

:::

## 方法三：最小速度/完成圆周条件

适用于判断粒子能否完成竖直圆周。

:::info[Steps]

1. 最高点：$mg + T = m\frac{v^2}{r}$
2. 临界条件：$T = 0$（绳子松弛前瞬间）
3. 代入得 $v_{\min} = \sqrt{gr}$
4. 结合能量守恒求起点所需速度：
   $v_{\text{起点}}^2 = v_{\text{最高点}}^2 + 4gr$

:::

## 方法四：变速率竖直圆周

适用于速度变化导致向心力变化的问题。

:::info[Steps]

1. 写能量守恒找出速度与角度的关系
2. 向心力方程：$T - mg\cos\theta = m\frac{v^2}{r}$
3. 其中 $\theta$ 从竖直向下起算
4. 代入 $v^2$ 表达式求 $T(\theta)$

:::
