# Mark Scheme Patterns — Circular Motion

## 典型给分点分布

| 步骤 | 标记 | 说明 |
|------|------|------|
| 写出向心力公式 | **M1** | $F = m\frac{v^2}{r}$ 或 $mr\omega^2$ |
| 正确确定半径 | **M1** | 如 $r = L\sin\theta$ |
| 竖直分量方程 | **M1** | 如 $T\cos\theta = mg$ |
| 水平分量方程 | **M1** | 如 $T\sin\theta = mr\omega^2$ |
| 角速度/线速度换算 | **B1** | $v = r\omega$ |
| 能量守恒 | **M1** | 求不同高度速度 |
| 最高点临界条件 | **M1** | $v \geq \sqrt{gr}$ |
| 最终数值 | **A1** | 3 s.f. |

## 常见给分模式

- 圆锥摆：竖直 **M1** → 水平 **M1** → 联立 **M1** → $\theta$ 或 $\omega$ **A1**
- 竖直圆周最低点：向心力 **M1** → 代入 **M1** → $T$ **A1**
- 完成圆周条件：能量守恒 **M1** → 临界速度 **M1** → 答案 **A1**
