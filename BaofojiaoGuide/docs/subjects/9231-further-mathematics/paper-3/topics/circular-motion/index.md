# Circular Motion（圆周运动）

## 考纲要求

- 理解向心加速度 $a = \frac{v^2}{r} = r\omega^2$
- 水平圆周运动（圆锥摆、倾斜轨道）
- 竖直圆周运动
- 确定完成竖直圆周所需的最小速度

## 核心公式

| 公式 | 说明 |
|------|------|
| $a = \frac{v^2}{r}$ | 向心加速度（线速度形式） |
| $a = r\omega^2$ | 向心加速度（角速度形式） |
| $F = m\frac{v^2}{r}$ | 向心力 |
| $v = r\omega$ | 线速度与角速度关系 |
| $T = \frac{2\pi}{\omega}$ | 周期 |
| $v_{\min} = \sqrt{gr}$ | 竖直圆周最高点最小速度 |

## 常见问题类型

1. 水平圆周运动（张力/反力计算）
2. 竖直圆周运动（各点速度、张力）
3. 圆锥摆周期
4. 倾斜轨道（banked track）

## 解题步骤

1. 确定圆周半径 $r$
2. 确定向心力来源
3. 列 Newton 第二定律：$F = m\frac{v^2}{r}$
4. 如有需要，结合能量守恒
