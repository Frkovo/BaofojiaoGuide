# 考前速记 — Circular Motion

## 核心公式

| 公式 | 说明 |
|------|------|
| $a = \frac{v^2}{r} = r\omega^2$ | 向心加速度 |
| $F = m\frac{v^2}{r} = mr\omega^2$ | 向心力 |
| $v = r\omega$ | 线速度与角速度 |
| $v_{\min} = \sqrt{gr}$ | 最高点最小速度（绳子） |
| $T = 2\pi\sqrt{\frac{L\cos\theta}{g}}$ | 圆锥摆周期 |

## 易错点

- 圆锥摆 $r = L\sin\theta$（不是 $L$）
- 竖直圆周用能量守恒求速度
- 绳子 vs 杆：绳子最高点要求 $v \geq \sqrt{gr}$
- 向心力不是独立力，是合力分量

## 解题策略

1. 确定半径
2. 列向心力方程
3. 列竖直/能量方程
4. 联立求解
