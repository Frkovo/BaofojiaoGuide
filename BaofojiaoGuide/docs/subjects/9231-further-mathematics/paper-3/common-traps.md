# Paper 3 常见陷阱

## 10 大陷阱详解

### Trap 1: 抛射运动中混淆 $v_y$ 与 $v$

| 项目 | 内容 |
|------|------|
| **描述** | 求某时刻速度时直接用 $v = u + at$ 得到的是竖直方向速度 $v_y$，而非合速度 |
| **后果** | 合速度算错，后续动能计算全错 |
| **正确做法** | 合速度 $v = \sqrt{v_x^2 + v_y^2}$，其中 $v_x = V\cos\theta$ 不变 |

### Trap 2: 力矩计算中力臂取错

| 项目 | 内容 |
|------|------|
| **描述** | 取矩时用力的作用点到矩心的直线距离而非垂直距离 |
| **后果** | 力矩方程错误，平衡条件不满足 |
| **正确做法** | 取矩 = 力 $\times$ 从矩心到力作用线的**垂直距离**，或分解力后取分量 |

### Trap 3: 圆周运动中把向心力当成独立力

| 项目 | 内容 |
|------|------|
| **描述** | 在受力图中额外画一个"向心力" |
| **后果** | 力的数量多了一个，方程多一项无法求解 |
| **正确做法** | 向心力是合力而非独立力，受力图只画真实力（张力、重力、法向力、摩擦力） |

### Trap 4: 竖直圆周运动忘记能量守恒

| 项目 | 内容 |
|------|------|
| **描述** | 在竖直圆周底部和顶部之间直接用匀加速运动公式 |
| **后果** | 速度算错（圆周运动不是匀加速） |
| **正确做法** | 用能量守恒：$\frac12 mv_1^2 + mgh_1 = \frac12 mv_2^2 + mgh_2$ |

### Trap 5: 胡克定律中将 $x$ 与 $l$ 混淆

| 项目 | 内容 |
|------|------|
| **描述** | 弹力公式中使用 $T = \frac{\lambda}{l}\times(\text{current length})$ 而非 extension |
| **后果** | 弹力大小错误，整道题全错 |
| **正确做法** | Extension $x = \text{current length} - \text{natural length}$，弹力 $T = \frac{\lambda}{l}x$ |

### Trap 6: 弹性势能公式的系数混淆

| 项目 | 内容 |
|------|------|
| **描述** | 将 $\text{EPE} = \frac{\lambda x^2}{2l}$ 写成 $\frac12 kx^2$ 但不知 $k = \lambda/l$ |
| **后果** | 如果 $\lambda$ 和 $l$ 分别给出，直接写 $\frac12 kx^2$ 未体现参数关系，可能丢方法分 |
| **正确做法** | 用 $\frac{\lambda x^2}{2l}$ 或明确写出 $k = \lambda/l$ 后再用 $\frac12 kx^2$ |

### Trap 7: 变力问题中微分方程符号错误

| 项目 | 内容 |
|------|------|
| **描述** | 阻力方向与速度相反但写方程时符号弄反 |
| **后果** | $dv/dt$ 正负号错误，后续积分结果完全不对 |
| **正确做法** | 阻力总是与速度方向相反。向上为正时，阻力 $= -kv$；向下运动时 $v$ 为负，阻力也自动反向。先用矢量形式 $F = m\ddot{x}$ 再代入代数形式 |

### Trap 8: 动量守恒中速度方向符号不一致

| 项目 | 内容 |
|------|------|
| **描述** | 规定某一方向为正后，部分速度取了绝对值 |
| **后果** | 动量守恒方程不成立 |
| **正确做法** | 选定正方向后，所有速度带正负号代入 $m_1u_1 + m_2u_2 = m_1v_1 + m_2v_2$ |

### Trap 9: 斜碰中 NEL 应用方向错误

| 项目 | 内容 |
|------|------|
| **描述** | 在 oblique impact 中沿 line of centres 方向未正确使用 NEL |
| **后果** | 法向速度关系错误 |
| **正确做法** | 只取沿 line of centres 方向的速度分量应用 $e = \frac{v_2 - v_1}{u_1 - u_2}$，垂直方向速度不变 |

### Trap 10: 多过程问题遗漏中间状态

| 项目 | 内容 |
|------|------|
| **描述** | 题目涉及抛射 + 碰撞 + 圆周运动等过程，只算了前一半 |
| **后果** | 少一大半分值 |
| **正确做法** | 读完题先在草稿上列出所有阶段（e.g. 抛射 → 碰撞 → 圆周运动），标出各阶段的已知量和未知量，再逐个击破 |
