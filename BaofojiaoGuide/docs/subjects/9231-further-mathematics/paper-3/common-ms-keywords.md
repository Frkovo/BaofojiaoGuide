# Mark Scheme 关键词

:::info[MS Keywords]
了解评分关键词有助于针对性答题，确保不丢分。
:::

## 动词指令

| 关键词 | 含义 | 常见分值 |
|--------|------|---------|
| **Find** | 求值，展示推导过程 | 2–4 |
| **Show that** | 证明已知结果，必须展示完整推导 | 3–5 |
| **Determine** | 确定值或状态，需论证 | 2–4 |
| **State** | 直接给出结果，无需推导 | 1 |
| **Explain** | 解释原因或原理 | 1–2 |
| **Sketch** | 作图标注关键值 | 1–2 |
| **Verify** | 验证某个结论成立 | 2–3 |
| **Hence** | 使用上一问的结果继续推导 | 变化 |

## Mark Point 标记含义

| 标记 | 含义 | 说明 |
|------|------|------|
| **M1** | Method mark | 正确的方法，即使计算错误也可能给 |
| **A1** | Accuracy mark | 正确的答案，依赖于前面的 **M** 分 |
| **B1** | Independent mark | 独立给分，不依赖其他步骤 |
| **DM1** | Dependent method | 依赖于上一步的结果 |
| **ft** | Follow through | 前面错误但此处方法正确可给分 |

## 常见给分点

- 建立坐标系并标示方向 → **B1**
- 写出正确的运动方程 → **M1**
- 代入初始条件 → **M1**
- 正确的数值答案 → **A1**
- 单位正确 → **B1**
- 精度（3 s.f.）→ **B1**

:::tip[Pro Tip]
即使不会算也可以得 method marks。写出正确公式、代入已知值、展示步骤。
:::
