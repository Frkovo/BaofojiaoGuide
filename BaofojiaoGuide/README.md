# 抱佛脚 Guide

> **CIE IGCSE / AS / A Level 临时抱佛脚复习指南**

本网站使用 [Docusaurus](https://docusaurus.io/) 构建。

## 安装

```bash
npm install
```

## 本地开发

```bash
npm start
```

启动本地开发服务器，打开浏览器访问 `http://localhost:3000`。

## 构建

```bash
npm run build
```

生成静态文件到 `build/` 目录。

## 部署

使用 SSH：

```bash
USE_SSH=true npm run deploy
```

不使用 SSH：

```bash
GIT_USER=<你的 GitHub 用户名> npm run deploy
```
